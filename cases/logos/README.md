## ⚠️ Чого бракує (gaps)
1. **Imigrant** — повністю порожній блок (див. вище).
2. **Інші клієнти з кейсів** — у папці `cases/` ~19 кейсів (каякінг, салон краси, готель для тварин, забудовники, школи англійської, інтернет-магазин, виробництво тощо), але **логотипів цих клієнтів немає**. Якщо потрібні для портфоліо — треба зібрати лого + дозвіл на згадку.
3. **Формат:** бажано додавати лого у `.png`/`.svg` на прозорому фоні + 1 рядок опису (сфера, сайт) — щоб одразу йшло на сайт/в презентацію.
