## ⚠️ Ключові обмеження

| Система | Головне обмеження |
|---------|-----------------|
| **HubSpot** | EN інтерфейс + висока ціна → непрактичний для UA малого бізнесу |
| **Pipedrive** | EN інтерфейс; слабкий для e-commerce/роздробу, немає ERP |
| **Odoo** | Потрібен технічний партнер ($5K–$25K впровадження); без партнера ризик |
| **Perfectum** | Більше ERP ніж CRM — надмірний для малого сервісного бізнесу |
| **NetHunt** | Тільки в Gmail / Google Workspace екосистемі |
| **Planfix** | Складне первинне налаштування; потребує адміна або консультанта |
| **KeyCRM** | Flat $19 + опції можуть рости; слабший PM-функціонал |
| **KeepinCRM** | Обмежений ERP-функціонал; не підходить для виробництва/оптового складу |
