# HubSpot — по яких нішах підходить

> Зворотний індекс: замість "яка CRM для ніші X" → "для яких ніш підходить HubSpot"
> ✅ Перевірено: червень 2026

## Ключові факти
- **Сайт:** hubspot.com
- **Ціна:** $0 (Free) → $15/user (Starter) → $90/user (Sales Hub Pro) → $150/user (Enterprise)
- **UA мова:** ❌ (інтерфейс EN; сайт частково UA)
- **Юрисдикція:** 🇺🇸 (Nasdaq-listed, GDPR-compliant)
- **Особливість:** Marketing + Sales + Service Hub в єдиній екосистемі; Breeze AI на борту

## ✅ Підходить для (топ-ніші)

| Ніша | Роль у ніші | Позиція | Файл |
|------|------------|---------|------|
| 💻 SaaS / software | Топ-1: Breeze AI, sequences, Chargebee-native, MRR/ARR через інтеграції, Customer Agent | Лідер ($1M–$10M ARR) | [saas.md](../niches/saas.md) |
| 💻 IT / Digital-агентства | Топ-1 для великих IT-аутсорс ($1M+ ARR): повний стек маркетинг + sales + support | Лідер (Enterprise) | [it-digital.md](../niches/it-digital.md) |
| 💼 B2B-послуги / консалтинг | Топ-2: sequences, Jira-native через Zapier, AI scoring; для рекрутингових агентств | Сильна альтернатива | [b2b-services.md](../niches/b2b-services.md) |

## ⚠️ Можна, але є кращі варіанти

| Ніша | Чому другий вибір |
|------|-----------------|
| 🏠 Нерухомість | Немає шахматки, немає LUN/DOM.RIA нативно; Plektan/LigaPro краще |
| ⚖️ Юристи | Немає LPM-функцій; DiLegal або Jusnote для юридичної ніші |
| 🎓 Освіта | Дорого і складно для UA-освітніх центрів; NetHunt або KeepinCRM простіше |

## ❌ Не підходить
- **🛒 E-commerce (UA)** — немає нативних Prom/Rozetka/Nova Poshta; KeyCRM лідер
- **🏥 Медицина / МІС** — не є МІС; немає ЕСОЗ, онлайн-запису
- **🔧 Автосервіс** — немає заказ-нарядів, VIN-карток
- **💅 Б'юті / салони** — немає розкладу майстра, онлайн-запису
- **🍽️ Ресторани** — не є POS
- **🏭 Виробництво** — немає BOM, MRP — Odoo або Perfectum
- **📦 Опт / дистрибуція** — немає ERP-складу, партіонного обліку
- **💰 МФО / банки** — немає LOS/LMS, скорингу УБКІ
- **🚛 Логістика** — немає TMS-функціоналу
- **Малий бізнес UA** — EN інтерфейс + висока ціна роблять непрактичним для більшості UA SMB

## Типовий клієнт
IT-аутсорс або SaaS-компанія з міжнародними клієнтами ($1M+ ARR), яка продає англомовній аудиторії, веде inbound маркетинг (SEO/контент) і потребує єдиного стеку: Marketing Hub (ліди) + Sales Hub (угоди) + Service Hub (support). Breeze AI прискорює prospect research і написання email. Часто вибирають коли переросли Pipedrive або NetHunt.
