# Odoo — по яких нішах підходить

> Зворотний індекс: замість "яка CRM для ніші X" → "для яких ніш підходить Odoo"

## Ключові факти
- **Сайт:** odoo.com
- **Ціна:** Community — безкоштовно (open-source); Enterprise — від €19.9/user/міс; SaaS — від €11.9
- **UA мова:** ⚠️ часткова локалізація (інтерфейс перекладено, але UA-бухгалтерія потребує партнера)
- **Юрисдикція:** 🇧🇪 (Бельгія, відкритий код)
- **Особливість:** Повноцінна open-source ERP з CRM, склад, бухгалтерія, виробництво, HR, eCommerce — все в одному

## ✅ Підходить для (топ-ніші)

| Ніша | Роль у ніші | Позиція | Файл |
|------|------------|---------|------|
| 🏭 Виробництво / ERP | Топ-1: BOM, MRP, рецептури, lot-tracking, маршрути; UA-партнери є | Лідер | [manufacturing.md](../niches/manufacturing.md) |
| 📦 Опт / дистрибуція | Топ-2 (після Perfectum для великих): ЗЕД, B2B-портал нативно, AI-попит, мультисклад | Лідер (середній/великий) | [wholesale.md](../niches/wholesale.md) |
| 🏗️ Будівництво / підрядники | Гнучкий ERP для складних підрядників; кастомізація під кошториси та поетапні оплати | Альтернатива | [construction/00-overview.md](../niches/construction/00-overview.md) |
| 🛒 E-commerce | Full ERP для великого e-com: склад + бухгалтерія + CRM; Community free — варіант старту | Альтернатива (ERP-рівень) | [ecommerce.md](../niches/ecommerce.md) |

## ⚠️ Можна, але є кращі варіанти

| Ніша | Чому другий вибір |
|------|-----------------|
| 💼 B2B-послуги / консалтинг | Odoo надто "важкий" для сервісного бізнесу; Planfix або NetHunt легші та дешевші |
| 💻 IT / Digital-агентства | Немає CRM+PM-зв'язки з email-sequences та LinkedIn; краще NetHunt або Pipedrive |
| 💰 Фінанси / МФО | Є бухгалтерський модуль, але немає LOS/LMS і скорингу УБКІ |

## ❌ Не підходить
- **💅 Б'юті / салони краси** — немає онлайн-запису, розкладу майстрів, касовий модуль не адаптований
- **🏥 Медицина / МІС** — немає ЕМК, ЕСОЗ (eHealth) інтеграції з НБУ; потрібні Cliniccards/Health24
- **🍽️ Ресторани / HoReCa** — є POS-модуль, але Poster POS значно простіший та дешевший для UA ресторану
- **🚛 Логістика / TMS** — базовий склад є, але TMS-функцій (рейси, CMR, GPS водіїв) немає; краще 4logist
- **💻 SaaS / software** — немає PLG-архітектури, MRR/churn трекінгу; краще Attio або HubSpot
- **Малий бізнес без IT-ресурсу** — впровадження $5K+; без партнера запустити складно

## Типовий клієнт
Виробниче або дистрибуційне підприємство (меблі, харчове виробництво, оптова торгівля) з 20–200 співробітниками, яке переходить з 1С/BAS і потребує ERP-рівня: BOM, MRP, склад, бухгалтерія, CRM — все в одній системі. Обирає Odoo через відкритий код, відсутність вендор-локу та наявність UA-партнерів з впровадження.
