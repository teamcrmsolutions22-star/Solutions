# Perfectum CRM+ERP — по яких нішах підходить

> Зворотний індекс: замість "яка CRM для ніші X" → "для яких ніш підходить Perfectum"
> ✅ Перевірено: червень 2026

## Ключові факти
- **Сайт:** perfectum.ua
- **Ціна:** від 350 грн/user/міс (хмара) + коробкова версія
- **UA мова:** ✅
- **Юрисдикція:** 🇺🇦 (повністю українська система)
- **Сертифікат:** ISO 27001 ✅
- **Особливість:** CRM + ERP в одному — склад, документи, специфікації, ПРРО, фінанси

## ✅ Підходить для (топ-ніші)

| Ніша | Роль у ніші | Позиція | Файл |
|------|------------|---------|------|
| 📦 Опт / дистрибуція | Топ-1: склад, прайси по дилерах, кредитні ліміти, ПРРО, дебіторка в одному | Лідер | [wholesale.md](../niches/wholesale.md) |

| 💰 Фінанси / МФО | Топ-3: кредитний облік, фінансові потоки — для малих МФО | Альтернатива | [finance/00-overview.md](../niches/finance/00-overview.md) |
| 💼 B2B-послуги | ERP-рівень для послугових компаній із складом або документообігом | Нішева альтернатива | [b2b-services.md](../niches/b2b-services.md) |

## ⚠️ Можна, але є кращі варіанти

| Ніша | Чому другий вибір |
|------|-----------------|
| 🛒 E-commerce | Є модуль маркетплейсів, але KeyCRM чи KeepinCRM простіше і дешевше для старту |
| 💻 IT / Digital | Надто "важкий" ERP для IT-агентства — Planfix або NetHunt простіші |
| 💼 Консалтинг | Більше ERP ніж CRM — Planfix або HubSpot ефективніше |

## ❌ Не підходить
- **🍽️ Ресторани** — не є POS; Poster POS для HoReCa
- **💅 Б'юті / салони** — немає онлайн-запису майстра, розкладу
- **🏋️ Фітнес** — немає абонементів, заморозки
- **🏥 Медицина** — не є МІС; немає ЕМК, ЕСОЗ
- **💻 SaaS** — надто операційна система, немає PLG/MRR/ARR трекінгу

## Типовий клієнт
Малий або середній виробничий або дистрибуційний бізнес у Україні (меблі на замовлення, харчовий цех, оптовик із 5–50 складськими позиціями), якому потрібне єдине рішення замість 1С/BAS: склад, замовлення, документи, CRM і ПРРО — усе в одному UA-продукті за 350 грн/user.
