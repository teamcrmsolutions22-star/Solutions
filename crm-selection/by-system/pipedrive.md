# Pipedrive — по яких нішах підходить

> Зворотний індекс: замість "яка CRM для ніші X" → "для яких ніш підходить Pipedrive"
> ✅ Перевірено: червень 2026

## Ключові факти
- **Сайт:** pipedrive.com
- **Ціна:** $14/user (Essential) → $24 (Advanced) → $39 (Professional) → $49 (Power) → $79 (Enterprise)
- **UA мова:** ❌ (інтерфейс EN; підтримка EN)
- **Юрисдикція:** 🇪🇪/🇺🇸 (засновано в Естонії, NYSE-listed)
- **Особливість:** Activity-based selling; канбан-воронка; ~1000+ UA IT-компаній використовують

## ✅ Підходить для (топ-ніші)

| Ніша | Роль у ніші | Позиція | Файл |
|------|------------|---------|------|
| 💻 IT / Digital-агентства | Топ-2: ~1000 UA-компаній, activity-based selling, ClickUp нативно, Smart Docs + e-sign | Лідер | [it-digital.md](../niches/it-digital.md) |
| 💻 SaaS / software | Топ-4 для lean sales team: activity-based, Chargebee інтеграція, засновницькі продажі | Сильна альтернатива | [saas.md](../niches/saas.md) |
| 💼 B2B-послуги / консалтинг | Топ-2: Smart Docs + e-sign, ClickUp нативно, для digital-агентств та sales-teams | Сильна альтернатива | [b2b-services.md](../niches/b2b-services.md) |
| 🏠 Нерухомість | Загальна CRM для агентств без шахматки; налаштовується під угоди нерухомості | Нішева альтернатива | [real-estate.md](../niches/real-estate.md) |

## ⚠️ Можна, але є кращі варіанти

| Ніша | Чому другий вибір |
|------|-----------------|
| 🎓 Освіта | EN інтерфейс — незручно для UA-шкіл; NetHunt або KeepinCRM простіше |
| ⚖️ Юристи | Можна кастомізувати, але немає LPM (справи, процесуальні строки, billable hours) |
| 🏗️ Будівництво | Немає специфікацій BOM, кошторисів; Perfectum або KeyCRM краще |

## ❌ Не підходить
- **🛒 E-commerce (UA)** — немає нативних Prom/Rozetka/Nova Poshta; KeyCRM чи KeepinCRM
- **🏥 Медицина** — не є МІС; немає ЕСОЗ, онлайн-запису
- **🔧 Автосервіс** — немає заказ-нарядів, VIN-карток
- **💅 Б'юті / салони** — немає онлайн-запису, розкладу майстра
- **🍽️ Ресторани** — не є POS
- **🏭 Виробництво** — немає BOM, MRP
- **📦 Опт / дистрибуція** — немає ERP-складу, кредитних лімітів, партіонного обліку
- **💰 МФО / банки** — немає LOS/LMS
- **🚛 Логістика** — немає TMS
- **Малий UA-бізнес** — EN інтерфейс + ціна; KeepinCRM або Planfix практичніше

## Типовий клієнт
IT-аутсорс або digital-агентство з 5–50 людьми, яке продає міжнародним клієнтам (EN-мова), веде активний outbound через email або LinkedIn, потребує простої воронки + активностей + Smart Docs для підпису контрактів. Особливо популярний серед UA dev-shop-ів, де team lead або CEO виконує роль AE.
