## 🏢 ЧАСТИНА 2: Загальні UA CRM адаптовані для авто

### 16. KeepinCRM (keepincrm.com) ✅ — #1 для запчастин/автосалону

**Є окрема сторінка:** keepincrm.com/crm-for-car-showroom-and-service  
**Ціна:** Безкоштовно (1 user) / від **349 грн/user/міс** (прогресивна знижка: 10 users → 300 грн/user; 100+ users → 125 грн/user). Мінімальний період: **3 місяці** ⚠️

**Для авто-специфіки:**
- Картка авто з VIN/держномером та кількома авто на клієнта (через кастомні поля) ✅
- Заказ-наряди через угоди + шаблони документів ✅
- **Модуль «Записи»** — календар для фіксування запланованих візитів клієнтів з розподілом по кабінетах та майстрах ✅
- **Склад — найпотужніший серед UA CRM:**
  - Мультисклад (кілька філій) ✅
  - FIFO-собівартість ✅
  - Облік комплектів (автоматичне списання складників) ✅
  - Штрих-коди ✅
  - Модуль постачальників: мультивалюта, консолідовані закупівлі з угод ✅
- **Каса для офлайн магазину:** сканер штрих-кодів, фіскалізація Checkbox (ПРРО), зміни касирів ✅
- Гнучкі тригери для сезонних нагадувань: «через 180 днів від заміни масла → SMS» ✅
- Viber, Telegram, WhatsApp через інтеграції ✅
- Безліміт філій на Extended ✅

**Реальний кейс: AutoGroup (Одеса)** — провідний автомобільний холдинг, 17 років на ринку, мережа (Ford, Renault, Hyundai, Skoda, KIA, Citroen, Nissan, Mitsubishi, Peugeot). Мігрував з Bitrix24 та AutoCRM після 24.02.2022. Використовує KeepinCRM **3+ роки**. ✅

**Реальний розрахунок для малого СТО:** перший user безкоштовно → СТО з 3 співробітниками платить тільки за 2 ліцензії Extended = **698 грн/міс** ✅ (підтверджено джерелом Perplexity 2026)

**Обмеження:** Немає нативного VIN-каталогу; авто-специфіка через кастомне налаштування; мінімум 3 місяці підписки

### 17. KeyCRM (ua.keycrm.app) ✅ — для e-commerce запчастин

**Ціна:** $19/міс flat (безліміт users, 200 замовлень, **2 000 карток клієнтів**; +$9 за кожні 200 замовлень або 2 000 карток понадліміт; +$19 за кожне джерело синхронізації); 30 днів безкоштовно ✅

**Для авто-специфіки:**
- Сильний для **e-commerce запчастин**: мультисклад, маркетплейси (Prom/Rozetka), доставка (Nova Poshta) ✅
- Всі месенджери: Instagram, Telegram, Viber, WhatsApp, Facebook ✅
- Шаблони документів та заказ-наряди через замовлення ✅
- Підбір запчастин — ручне введення або імпорт прайсів ✅
- Кейс: магазин автозапчастин (e-commerce) ✅

**Обмеження:** Немає ресурсного календаря постів; немає авто-специфічних нормо-годин; більше для торгівлі ніж для сервісу

### 18. NetHunt CRM (nethunt.ua) ✅ — для автосалонів + сервісу

**Ціна:** від $24/user/міс (Basic); $48 Business; $96 Advanced (знижка 20% при річній оплаті)

**Для авто-специфіки (позиціонується на auto dealership):**
- Каталог ТЗ з комплектаціями ✅
- Облік тест-драйвів ✅
- **Trade-In** (обмін авто з доплатою) ✅
- Контроль кредитних та лізингових угод ✅
- Збір інформації про пробіг → прогноз наступного ТО → автоматичне завдання менеджеру або сповіщення у Viber/Telegram ✅
- Потужні email-послідовності ✅

**Обмеження:** Прив'язка до Gmail; немає нативного заказ-наряду; більше для автосалонів ніж СТО

### 19. Perfectum CRM+ERP (perfectum.ua) ✅

**Ціна:** Безкоштовна хмарна версія ✅; від **350 грн/user/міс** (платна); хмара для 5 users = **~2 160 грн/міс** ✅; Коробка: від **36 000 грн** (5 users, одноразово); Впровадження від 60 000 грн ⚠️

**Для авто-специфіки:**
- **START CRM:** ведення продажів послуг, лідогенерація з Facebook, договори, рахунки та телефонія ✅ (базовий вхід)
- **PROJECT-версія:** Scrum/Agile/Waterfall, **Діаграми Ганта** — ідеально для кузовних цехів з довгим ремонтом ✅
- **RETAIL-версія:** склад, маркетплейси, ПРРО Checkbox, POS, інтеграція з постачальниками ✅
- **ERP Enterprise:** об'єднує всі вищезгадані модулі для великого бізнесу ✅
- Кастомні поля для VIN, картка авто з історією ✅

**Обмеження:** Немає виокремленого галузевого рішення для СТО; потребує інтегратора; інтерфейс переважно RU/EN (UA — ⚠️)

### 20. OneBox OS (1b.app) ✅

**Ціна:** Free до 1000 records; від **$29/user/міс** (Cloud, мінімум 3 users → ~$87/міс); On-premise $699/user перший рік

**Для авто:** є шаблони заказ-нарядів, склад, онлайн-запис, фінанси, персонал — через партнерів (iCOLOR)  
**Обмеження:** Складне впровадження, потрібен інтегратор; вища ціна  
**Юрисдикція:** ⚠️ Зареєстрований в Україні, але є відомості про зв'язок з Білоруссю (🇺🇦/🇧🇾) — рекомендується уточнити перед впровадженням у КІІ

### 21. Uspacy (uspacy.com) ✅

**Ціна:** 392 грн/user/міс (Стандартний з 01.01.2026); 784 грн/user/міс (Професійний)

**Для авто:**
- Кастомні поля (VIN, марка, рік) ✅
- **AI-транскрипція дзвінків** → автоматичне резюме розмови + автоматичне **генерування задач із переписок/дзвінків** ✅
- Задачі та активності для ТО-нагадувань ✅
- Telegram, Viber, WhatsApp ✅
- Масові розсилки SMS/Viber для сезонних нагадувань ✅

**Обмеження:** Немає нативних заказ-нарядів, VIN-картотеки або планування ремонтних робіт. **Не рекомендується як основна CRM для СТО**

## 🌍 ЧАСТИНА 3: Міжнародні CRM

### 22. Zoho CRM ✅ — Automotive positioning + Zia AI

**Ціна:** від $14/user/міс (Standard); Zoho One $37/user/міс (all-in-one)  
**Automotive-специфіка:**
- Zoho має окрему сторінку "Automotive CRM" ✅
- Нативно: **CommandCenter** (multi-step processes), **Work Orders**, **VIN scanning**, **Dispatch Console** ✅
- **AutoGarage CRM** — marketplace app для Zoho (лише Enterprise) ✅
- **Zia AI:** рекомендації запчастин і ТО по історії покупок, скоринг лідів, прогноз конверсій ✅
- Автоматичні нагадування про сервісні візити та планові роботи ✅
- Мультивалюта (140+ валют) ✅
- SolidBrain Ukraine — офіційний TecDoc-партнер для Zoho-інтеграцій (⚠️ через окремий API, не нативно)
- Без UA інтерфейсу ⚠️

### 23. HubSpot CRM — автомотів positioning ⚠️ ДОРОГИЙ

**Ціна:** Free (базовий CRM) + Sales Hub Starter від $50/міс; Sales Hub Pro від $90/user  
**Automotive:** підходить для мережевих автосалонів; є Breeze AI ✅

**Юрисдикція:** 🇬🇪 Зареєстрована в Грузії (Pipedrive OÜ → куплена Vista Equity Partners, США); не є рос. продуктом ✅

### 25. Kommo (комерційно відділена від amoCRM) ⚠️ ОБЕРЕЖНО — РЕКОМЕНДУЄТЬСЯ УНИКАТИ

**Тип:** Міжнародна CRM-платформа (глобальний ребренд amoCRM)  
**AI-функції:**
- AI-чат-боти та автоматизація комунікацій ✅
- AI-інструменти для лід-менеджменту ✅

**Застереження:**
- **amoCRM (Kommo)** прямо описується в джерелах як **РОСІЙСЬКА CRM-система** — засновники й частина технічного стеку мають РФ-походження ⚠️
- Kommo комерційно позиціонується як окрема глобальна платформа, але **походження продукту і частина інфраструктури — від amoCRM (Росія)** ⚠️
- На відміну від Bitrix24 🚫🇷🇺 (позиціонує себе як міжнародна, але 1С-платформа / RU-коріння) — Kommo також не в реєстрі ДССЗЗІ поіменно; ризик обох залишається ⚠️
- Відсутня авто-специфіка (нема заказ-нарядів, VIN-картки, нормо-годин)

**Висновок:** Для автосервісу НЕ рекомендується ні за функціоналом, ні за юрисдикцією. Альтернативи: KeepinCRM, NetHunt, Uspacy.

**Для кого:** Мережеві СТО, автоімпортери, великі детейлінг-центри, де стандартного SaaS-функціоналу недостатньо

**Що включає:**
- Детальний бізнес-аналіз бізнес-процесів СТО ✅
- Розробка технічного завдання + унікальний UX для майстрів-приймальників ✅
- Програмування кастомного функціоналу (специфічні системи лояльності, інтеграція діагностичного обладнання) ✅
- Навантажувальне тестування безпеки ✅
- Навчання персоналу ✅
- Гарантує **абсолютну конфіденційність** комерційних даних (власні сервери) ✅

**Ціна:** ⚠️ Індивідуальний прорахунок (значно дорожче SaaS); підходить для бюджету від ~$20 000+  
**Юрисдикція:** Україна ✅
