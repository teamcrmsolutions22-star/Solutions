## ⚡ Ключове розмежування: б'юті ≠ класична CRM-воронка

```
Ніша              |  Ядро автоматизації                          |  Чого НЕ потрібно
------------------|----------------------------------------------|------------------
Б'юті (B2C)       |  Онлайн-запис + графік майстрів + retention  |  Лінійна воронка продажів
                  |  + боротьба з no-show                        |  Ліди / pipeline / B2B-комунікації
```

**Ключові метрики б'юті-бізнесу:** Retention Rate, LTV клієнта, завантаженість майстрів/кабінетів, частота повернень, no-show rate.

**Наслідок:** Загальні CRM (KeyCRM, KeepinCRM, Kommo 🚫🇷🇺, Bitrix24 🚫🇷🇺) **не закривають б'юті-сценарій** без суттєвого доопрацювання — нема нативного віджета запису до майстра, ресурсного планування кабінетів, медкарти з формулами фарбування.

## 🏆 Топ-3 рекомендації по сегментах

### Сегмент A: Барбершоп / Перукарня (1–10 крісел)

**Ключові потреби:** швидкий онлайн-запис, walk-in + online, мінімум кроків для клієнта, телефонія, нагадування, простий старт

1. **EasyWeek** — автоматична генерація сайту, Google Reserve, 3–4 кроки до броні, безкоштовний тариф для старту (30 записів/міс), mobile-friendly
2. **Bookon CRM** — найкраща нативна IP-телефонія (Binotel): при вхідному дзвінку бачиш картку клієнта і його останній стриж; тариф від 375 грн/міс
3. **Appointer** — простий старт, QR-оплата Monobank/Privat для підтвердження броні, Checkpoint ПРРО, 560 грн/міс (річна)

### Сегмент B: Салон краси повного циклу / Мережа (5–40 майстрів, 2–10 філій)

**Ключові потреби:** паралельне бронювання кабінетів і апаратів як ресурсів, мультифілія з RBAC, складні схеми ЗП для різних напрямів (волосся / нігті / косметологія), єдина клієнтська база

1. **CleverBOX:CRM** — єдина система, що управляє кабінетами/апаратами як ресурсами без накладок; white-label клієнтський app; каскадні Viber+TG+SMS нагадування; від €42/міс per branch
2. **Beauty Pro CRM** — 150+ звітів, складні схеми ЗП для мережі, network discount, активно оновлюється (березень 2026), від ~$21/міс
3. **EasyWeek (Unlimited)** — €49.90/міс, безлімітний персонал, підходить для brand-scale мережі; Google Reserve live для Ukraine

### Сегмент C: Нігтьова студія / Соло-майстер (бюджет)

**Ключові потреби:** мінімальна ціна, Instagram-запис, абонементи на курс манікюру, прив'язка до конкретного майстра

1. **EasyWeek Free** — 1 майстер, 30 записів/міс безкоштовно; платний Solo від €12.50/міс (~520 грн); готовий сайт-візитка; Google Reserve

**Ключові потреби:** кабінети і масажні столи як ресурс, довгі процедури (блокування часу), передоплата/депозит, подарункові сертифікати, абонементи
