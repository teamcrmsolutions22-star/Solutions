> **Сегменти:** МФО / мікрокредитування · банківська CRM · страхування (агентська мережа) · обмін валюти · платіжні агенти / BNPL · інвестиції / брокери · фінансовий аутсорсинг  
> **Джерела:** 5 паралельних deep research агентів (червень 2026) + 4 uploaded research docs (2× Perplexity + ChatGPT Deep Research): creatio.com, microf1.ua, crm.ua, mahaon.ua, lbs.systems, insk.com.ua, ukasko.ua, insurs.online, ususoft.com.ua, bank.gov.ua, ubki.ua, diia.gov.ua, gigacloud.ua та ін.

## ⚡ TL;DR — Швидкий довідник

| Клас системи | Для кого | Ключові системи |
|-------------|---------|----------------|
| LOS/LMS | МФО, ліцензовані кредитори | MicroF1, Lendex (Artjoker), Mahaon, LBS Cloud |
| Enterprise CRM/BPM | Банки, великі ФК | Creatio (Banking/Lending), CRM.ua XRM Banking (MS Dynamics 365), Salesforce FSC |
| SMB CRM для ФК | МФО малі, КС, ломбарди, аутсорс | Perfectum (мікрокредит-модуль), OneBox (УБКІ-конектор) |
| Загальні CRM | Фронт-офіс поверх core | KeyCRM, KeepinCRM, NetHunt, Uspacy |
| Страхові агентські портали | Страхові агенти, брокери | Ukasko.connect, Insurs Online, Polis.ua B2B |
| Страхова core | Тільки INSK (не продається) | INRISK (insk.com.ua) — внутрішній продукт |
| Обмін валюти | Обмінники, платіжні агенти | AS-Service, USU (⚠️ due diligence), Perfectum, KeyCRM |
| Хмарна інфраструктура | Всі ФУ під НБУ | GigaCloud (КСЗІ+ISO), De Novo |

**Ключовий висновок:** Для МФО/мікрокредиторів **CRM без LOS/LMS не закриває задачу**. Скоринг, кредитний конвеєр, нарахування відсотків і колекшн живуть у спеціалізованих системах (MicroF1, Lendex, Mahaon). CRM — лише sales-шар поверх core. Для банківського enterprise CRM — потрібні Creatio, XRM Banking (MS Dynamics), Salesforce FSC.

## ⚡ Ключове розмежування: CRM ≠ LOS/LMS ≠ страхова core-система

```
Клас               |  Що вміє                                                |  Кому підходить
-------------------|----------------------------------------------------------|-----------------
CRM (front-office) |  Ліди, воронка, комунікації, активності, pipeline        |  Будь-яка ФК як доповнення
LOS                |  Loan Origination: анкета → скоринг → рішення → видача   |  МФО, ліцензовані кредитори
LMS                |  Loan Management: нарахування %, графік платежів, NPL     |  МФО, банки, лізинг
Collection CRM     |  NPL-колекшн: стадії, автодзвінки, суд, реструктуризація |  МФО + банки
Insurance Core     |  Тарифи, поліси, збитки, перестрахування, регул. звітність | Страховики
Agent Portal       |  Порівняння тарифів, продаж полісів агентами/брокерами   |  Агентські мережі, СК
Обмін валют        |  Каса, курси, Form 377, мультивалюта                     |  Обменники, банки
Enterprise CRM/BPM |  Кредитний конвеєр, 360° клієнт, BPM, AI-агенти          |  Великі банки / ФК
```

## ⚡ Регуляторна карта: що потрібно для роботи ліцензованої ФУ

```
Вимога                         | Хто підпадає             | Дедлайн
-------------------------------|--------------------------|------------------
НБУ Постанова №99 (хмари)      | Банки + небанківські ФУ  | 01.05.2026
НБУ Постанова №143 (СУІБ)      | Небанківські ФУ (МФО...) | 13.12.2026
ДССЗЗІ — заборона 1С/BAS       | Всі держоргани + регул.  | Вже чинно з 09.01.2026
Санкції РНБО (Bitrix24)        | Всі юр. особи в UA       | До 2033
Закон ВРУ — Kommo/amoCRM       | Всі юр. особи в UA       | З 19.08.2024
```

**Чек-ліст для ФУ перед вибором CRM/системи:**
- [ ] Юрисдикція зберігання даних: UA / EU / UK / USA / Canada?
- [ ] КСЗІ + ISO 27001 у хмарного провайдера?
- [ ] Система не в переліку ДССЗЗІ (09.01.2026)?
- [ ] Система не під санкціями РНБО (Bitrix24) або законом ВРУ (Kommo)?
- [ ] Є УБКІ-інтеграція (для кредиторів)?
- [ ] Є підтримка Diia.Підпис / BankID для e-KYC?
- [ ] Договір з провайдером має 18 обов'язкових пунктів згідно Постанови №99?
