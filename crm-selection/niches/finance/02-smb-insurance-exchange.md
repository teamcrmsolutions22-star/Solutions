## 💼 ЧАСТИНА 3: CRM для SMB у фінансах (МФО, КС, ломбарди, страхові агентства)

### 7. Perfectum CRM+ERP ✅ — мікрокредит-модуль

**Тип:** Українська CRM+ERP зі спеціалізованим модулем для мікрофінансів  
**Мікрокредит-модуль включає:**
- Кредитний калькулятор (ануїтет, диференційований)
- Платіжний календар — графік платежів автоматично
- Відстеження прострочень + сповіщення
- Документи (договір, заявка, графік)

**Ціна:**
- Cloud: від ~350 грн/user/міс (~1500 грн/міс мінімум)
- Base plan (cloud): від ~1000–1500 грн/міс (для невеликих команд)
- Коробка (On-premise): від ~36 000 грн

**Модуль циклічного білінгу (для фін. аутсорсингу):** автоматичне створення рахунків за розкладом (щомісяця, щокварталу) ✅

**Юрисдикція:** ✅ Україна (perfectum.ua)  
**Для кого:** МФО малого розміру, кредитні спілки, ломбарди — де не потрібна повноцінна LOS-платформа

---

### 8. OneBox OS ✅ — УБКІ-конектор нативний

**Тип:** CRM + ERP-конструктор (Ukrainian low-code платформа)  
**Унікальна фіча для фінансів:**  
- **УБКІ-конектор нативний** — Report 10 (кредитний звіт фіз. осіб) + Report 25 (юридичні особи) ✅
- Страхові конфігурації (через marketplace додатків)
- BPM (бізнес-процеси довільної складності)

**Ціна:** $29/user/міс (мінімум 3 users); або індивідуальне ліцензування для складних проєктів  
**Юрисдикція:** ✅ Україна (odbox.ua)  
**Для кого:** МФО / фінансові агентства / страхові агентства — де потрібний УБКІ + кастомні BPM

### 9. LBS Cloud (lbs.systems) ✅

**Тип:** CRM + ERP з банківським модулем  
**Особливість:** Один із небагатьох Ukrainian SaaS із вбудованим банківським/фінансовим модулем (облік позик, відсотки, залишки)  
**Юрисдикція:** ✅ Україна  
**Ціна:** ⚠️ Публічного прайсу немає; уточнювати у lbs.systems

### Інші SMB-CRM для фронт-офісу у фінансах

**KeyCRM** — Омніканал, месенджери, базовий фін. облік; кредитна математика — в окремій core; підходить для фронт-офісу МФО/лізингу  
**KeepinCRM** — CRM + завдання + месенджер; базовий фіноблік для невеликих аутсорсингових команд  
**NetHunt** — Gmail-орієнтований; комунікація та погодження документів прямо з Gmail / Google Workspace  
**Uspacy** — CRM + завдання + месенджер; EU-хмара (GDPR); підходить для невеликих фін. аутсорсерів  
**HubSpot** — CRM для фінансових радників: контакти, воронки, Gmail, зустрічі; портфельний облік — в окремих платформах або BI

## 🛡️ ЧАСТИНА 4: Страхування — агентські платформи та B2B-портали

### 10. Ukasko.connect ✅ — #1 B2B-платформа для страхових агентів/брокерів

**Тип:** B2B SaaS агентська платформа  
**Що включає:**
- Порівняння тарифів ОСЦПВ від усіх СК онлайн
- База даних МВС (перевірка авто за номером) ✅
- **Diia** — автозавантаження даних з ДП ✅ + **автоматичне вивантаження полісів у застосунок «Дія»** ✅
- Калькулятор + моментальне оформлення полісу
- Автоматичне адміністрування комісійних винагород агентів ✅
- Контроль взаєморозрахунків ✅
- B2B-кабінет для агентів / брокерів
- Відповідає вимогам до страхових посередників та брокерів ✅

**Ціна:** SaaS-підписка / комісія. Уточнювати у ukasko.ua.  
**Для кого:** Страхові агенти, брокери, автодилери що продають ОСЦПВ, партнерські мережі СК  
**Юрисдикція:** ✅ Україна

### ⚠️ ІНРISK / INSK — ВАЖЛИВА ВІДМОВА + ТЕХНІЧНІ ДЕТАЛІ

> **INRISK (insk.com.ua)** — це власний внутрішній продукт страхової компанії INSK (Київ).  
> **НЕ продається** стороннім страховикам як платформа.  
> Якщо клієнт шукає "страхову core-систему" — INRISK ≠ опція для них.  
> Є окремий домен inrisk.com.ua (система входу), не пов'язаний з insk.com.ua.

**Технічний функціонал INRISK (для розуміння рівня core-систем):**
- Продаж страхових договорів із автоматичним розрахунком вартості при розірванні ✅
- Workflow врегулювання страхових подій: від онлайн-прийому заяви до виплати ✅
- Автоматичні виплати до **15 хвилин** після затвердження ✅ (підтверджено CHI Software)
- Особистий кабінет клієнта + інтеграція з банками ✅
- Відповідність стандартам регулювання страхового ринку України ✅
- Продукт «Все включено» (комплексне страхування) реалізовано на цій платформі

## 💱 ЧАСТИНА 5: Обмін валюти — спеціалізовані системи

### 13. USU (ususoft.com.ua) ⚠️ — лідер ринку, але перевіряти юрисдикцію

**Тип:** Спеціалізована система для обмінних пунктів валют  
**Що включає:**
- Мультифіліальність (один акаунт — багато обмінних пунктів)
- Автоматичне формування **Форми 377 НБУ** (обов'язкова звітність для обмінників) ✅
- Курсова аналітика, прибуток/збитки по парах
- Касові операції, інкасація
- Ролі: касир / керівник

**⚠️ Ризик:** Платформа-основа (стек розробки/корені продукту) має ознаки РФ-спадщини. Рекомендується перевірити актуальний стан юрисдикції перед впровадженням у фінустанові під наглядом НБУ.  
**Юрисдикція:** Формально Україна, але потребує due diligence  
**Ціна:** Ліцензійна, per filial. Уточнювати.
