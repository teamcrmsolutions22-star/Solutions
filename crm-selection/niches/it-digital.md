## ⚡ Контекст ніші: чому IT/Digital — окремий випадок

**Ключові відмінності від звичайного B2B:**

| Параметр | IT-аутсорс | Digital-агентство | Стандартний B2B |
|----------|-----------|------------------|-----------------|
| Цикл угоди | 1–9 місяців | 2–8 тижнів | 1–4 тижні |
| Модель | T&M / Fixed Price / Retainer | Проєкт + ретейнер | Разовий продаж |
| Валюта | USD / EUR (рахунки закордон) | USD / EUR / UAH | UAH |
| Документи | SOW / MSA / NDA / КП технічне | Бриф / КП / ретейнер-договір | Рахунок / договір |
| Стейкхолдери | CTO + CEO + Legal + Finance (зміна під час угоди) | 2–3 ЛПР | 1–2 ЛПР |
| CRM ↔ PM | Критично (Jira + CRM) | Важливо (ClickUp / Worksection) | Опціонально |
| Лідогенерація | Clutch / Upwork / LinkedIn / referral | Inbound / LinkedIn / referral | Різне |

**Основний запит від IT/Digital-компаній:**
1. Тривала угода → потрібне відстеження кількох стейкхолдерів
2. Ретейнер → авто-поновлення, контроль бюджету годин
3. CRM ↔ Jira/ClickUp/Worksection → передача після продажу без хаосу
4. Outbound → LinkedIn + email-sequences для cold outreach
5. T&M-інвойсинг → час → рахунок → мультивалюта

## ⚠️ ЧАСТИНА 7: Підводні камені кожного рішення

### HubSpot
1. **Ціна зростає нелінійно:** Free → Starter ($20) → Pro ($90/seat + $1,500 onboarding) = стрибок. Для команди 3 sales = $270/mo + $1,500 один раз.
2. **No Ukrainian UI** — проблема для команд де менеджери не читають англійськ.
3. **LinkedIn sequences** вимагають LinkedIn Sales Navigator Advanced Plus (~$1,300+/рік/user) — дорого.
4. **Marketing Hub окремо:** якщо потрібен inbound marketing, Marketing Hub Pro = $890/mo за 3 seats.

### Pipedrive
1. **Немає native time tracking:** T&M billing = потрібні Harvest/Toggl integration.
2. **Projects add-on:** $5/user/mo extra на Advanced+ — не включено базово.
3. **Email sequences лише від Growth ($39)** — на Lite ($14) немає.
4. **Smart Docs (SOW)** — лише Advanced+ ($39).

### NetHunt
1. **Немає invoicing і time tracking** — для IT-аутсорсу потрібен Zoho Invoice або FreshBooks поруч.
2. **Обовʼязково Google Workspace** — без Gmail/Google Apps не має сенсу.
3. **Sequences:** починаються з Business ($48/annual) — Basic ($24) без sequences.
4. **Pricing перевірити:** в різних джерелах різні цифри; офіційна сторінка nethunt.com/pricing є.

### Flowlu
1. **No Ukrainian UI** — RU + EN. Для UA команд що перейшли з Bitrix24 може бути незручно.
2. **PM = базовий Gantt:** складні спринти/Scrum не підтримує нативно (потрібен Jira окремо).
3. **Email sequences:** базові; для сильного outbound + Apollo/Lemlist.
4. **Телефонія:** native тільки Zadarma; UA АТС через Zapier.

### Scoro
1. **Мінімум 5 користувачів** — не для команди 2–3 осіб.
2. **Дорого:** від $26/u (Essential) до $71/u (Pro з billable timesheets).
3. **No Ukrainian UI.**
4. **Складне впровадження:** PSA-система вимагає налаштовки retainer templates, billing rules.

### Productive.io
1. **CRM-блок слабший** за спеціалізовані CRM.
2. **Email sequences відсутні** — потрібен окремий tool для outbound.
3. **No Ukrainian UI.**

### Close CRM
1. **Growth plan $99/user/mo** — дорого для невеликих UA команд.
2. **Не для project management** — тільки sales pipeline.
3. **Американський продукт:** USD тільки, no Ukrainian support.

### Attio
1. **Молодий продукт (2023–)** — менша екосистема інтеграцій.
2. **No invoicing, no PM** — потрібні сторонні tools.
3. **Орієнтований на SaaS-стартапи:** для класичного IT-аутсорсу може бути занадто "silicon valley".

### Uspacy
1. **Молодий продукт** (активний розвиток з 2023) — деякі функції автоматизації ще дозрівають.
2. **Немає time tracking / invoicing** — для T&M білінгу потрібні Zoho Invoice або FreshBooks.
3. **Email sequences відсутні** — тільки базова автоматизація через Process Builder.
4. **>200 users:** потенційно не вистачає глибоких аналітичних інструментів.
5. **Ціна зросла** з 1 січня 2026 (392 UAH/user/mo Standard).

### Planfix
1. **Висока складність:** self-service впровадження займе місяці — потрібен сертифікований інтегратор.
2. **Немає Jira/ClickUp native інтеграцій** — через API або Zapier.
3. **Зовнішня телефонія** (Ringostat/Binotel/UniTalk) — native немає, але добре інтегрується.
4. **Email sequences** — через Сценарії; потрібна налаштовка.

### Zoho One
1. **Складна екосистема:** 50+ продуктів у Zoho One = потрібен системний інтегратор (CRMOZ).
2. **No Ukrainian UI** нативно.
3. **Інтеграції між Zoho-продуктами** не завжди seamless.
