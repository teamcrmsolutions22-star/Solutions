## ⚡ Ключове розмежування: CRM ≠ ERP для виробництва

```
Клас           |  Що вміє                                     |  Кому достатньо
---------------|----------------------------------------------|------------------
CRM-only       |  Воронка продажів, клієнти, комунікації       |  Дистриб'ютори, сервіс
CRM + склад    |  +Товари, залишки, замовлення                 |  Торгівля, e-com
CRM + light ERP|  +Специфікація, базовий облік виробництва     |  Малий custom-order
ERP (MRP)      |  +BOM, MRP, маршрути, собівартість, MES       |  Будь-яке серійне або
               |                                              |  складне виробництво
```

**Ключовий висновок:** Для меблів, одягу, харчового або контрактного виробництва **чиста CRM не вирішує задачу**. Критично потрібна ERP-ланка: специфікації (BOM), виробничі замовлення, списання сировини, розрахунок собівартості. Більшість CRM (KeyCRM, KeepinCRM, Kommo, Uspacy) мають лише зачатки виробничого модуля.

> **Чому BAS ≠ безпечна заміна 1С:**
> BAS (Business Automation Software) базується на **BAF (Business Automation Framework)** — ліцензованій копії 1С:Підприємство 8.3. Власник торгової марки BAF — польська компанія NetHelp Jarocki Piotr, заснована ex-розробником польського офісу 1С. Схема з'явилась 2017–2018 як обхід санкцій РНБО проти 1С через European юрособу. Технічно — той самий 1С-код, ті самі об'єкти метаданих, та сама мова програмування. Станом на 09.01.2026 BAS ERP, BAS Бухгалтерія КОРП, BAS Управління торгівлею, BAS Документообіг КОРП є в переліку ДССЗЗІ. Джерела: innoware.com, EBA, ain.ua, oneservice-consulting.com.

## 🏭 ЧАСТИНА 1: ERP-системи з повноцінним виробничим контуром

### 1. Odoo Manufacturing (MRP) ✅ — #1 для українського SMB

**Тип:** Відкрита ERP (бельгійська, open-source)  
**Ціна:**
- **Community (self-hosted, open-source):** 💰 безкоштовна ліцензія; потрібне впровадження (від 30 000 грн за стартовий пакет ~20 год) ✅
- **Enterprise SaaS:** ~€14.80/user/міс (Standard) / €22.40/user/міс (Custom); Manufacturing module included ✅
- **UA-локалізація** від Self-ERP: €5.99/міс (річна) / €6.99/міс (щомісяця, до 50 users) ✅
- Ставка розробки UA-партнерів: 1 500–1 800 грн/год

**Виробничий модуль (MRP) — що вміє:**
- **BOM (специфікації):** багаторівневі, вкладення BOM у BOM (рецептура в рецептурі), автоматичне розкладання ієрархії ✅
- **MRP:** розрахунок потреб у матеріалах на основі замовлень + прогнозів + залишків ✅
- **MPS (Master Production Scheduling):** середньострокове планування ✅
- **Виробничі замовлення:** зі статусами, маршрутами, нарядами на роботи ✅
- **Робочі центри (Work Centers):** Діаграма Ганта, планування потужностей ✅
- **Розрахунок собівартості:** планова (при створенні) + фактична (після виконання: матеріали + час + накладні) ✅
- **Серійний/партійний облік, терміни придатності, штрих-коди** ✅
- **Якісний контроль** (QC checkpoints) ✅
- **PLM** (управління life-cycle продукту) ✅
- **MES-інтерфейс** для планшетів/мобільних у цеху ✅
- **Byproduct management** (побічна продукція, актуально для харчового) ✅

**Локалізація UA:** Інтерфейс українською ✅ | Бухоблік/ПДВ/ЗП — через партнерів (Self-ERP, ERP Ukraine) ✅

**Партнери в Україні (підтверджено):** Plugit (plugit.com.ua), Self-ERP (self-erp.com), ToDo LLC (todo.ltd), One Service Consulting (oneservice-consulting.com), CoDoo ERP, ERP Ukraine (erp.co.ua) ✅

**Кейси виробництва UA:**
- **Riot Division** — бренд одягу, інтеграція Shopify + Odoo MRP для виробництва на замовлення ✅
- **Yaroslav** — 4 фабрики (постільна білизна, ковдри, матраци), Odoo Manufacturing ERP ✅
- **Brave Technics** — виробництво електроніки/дронів (ToDo LLC) ✅
- Галузі UA-партнерів: текстиль, металообробка, меблі, харчове, пивоварні, корпоративні сувеніри ✅

**Підводні камені:**
- Community: безкоштовна ліцензія, але потрібна кастомізація; підходить для тих, хто готовий інвестувати в розробку
- Без UA-партнера впровадити складно; 6–12 місяців для повного виробничого проекту
- UA-бухгалтерська локалізація — окремий модуль/add-on, не «з коробки»

**Для кого ідеально:** SMB виробництво будь-якої галузі; особливо сильний для харчового (LOT/терміни), текстилю, контрактного B2B

### 2. IT-Enterprise ✅ — #1 для великого промислового виробництва UA

**Тип:** Українська ERP (IT-Enterprise LLC, Київ, засновано 1987)  
**Ціна:**
- **Cloud (IT-Enterprise.Cloud):** від **$299/міс** (компанія-пакет, не per-user) ✅
- **On-premise:** індивідуально; орієнтований на середній+ та великий бізнес
- Впровадження: від кількох місяців; типово 12–24 міс для повного промислового ERP

**Виробничий модуль — MRP II + MES + APS:**
- **MRP + MPS + APS** (Advanced Planning & Scheduling) — скінченна потужність, оптимізація послідовності ✅
- **MES** (Manufacturing Execution System) — реальний контроль цеху, диспетчеризація ✅
- Ресурсні специфікації (BOM) + Маршрутні карти (Routing) ✅
- Робочі центри, контроль потужностей ✅
- Нормативна + фактична собівартість ✅
- **EAM** (технічне обслуговування обладнання) ✅
- **SCM** — закупівлі, логістика ✅
- **PLM / R&D** ✅
- Якісний контроль ✅
- AI-функції в версії 2025 (ISO, GDPR, SOC2, SmartID) ✅
- 38 років на ринку, 1 250+ впроваджень

**Кейси (підтверджено):**
- **ФЕД** — авіаційне машинобудування, Київ (Industry 4.0)
- **Interpipe** — виробництво сталевих труб та залізничних коліс
- Галузі: металургія, машинобудування, харчове, хімічне, кабельне, фармацевтичне, агро

**Для кого:** Великі промислові підприємства (50+ співробітників виробництва); металообробка, машинобудування

## 🔧 ЧАСТИНА 2: CRM+ERP light — для малого custom-order виробництва

**Ціна:**
- Безкоштовна хмарна версія (базовий функціонал) ✅

**Виробничий модуль:**
- **Специфікація (наближений BOM):** вкладка «Специфікація» в проекті → список усіх матеріалів/виробів для виконання замовлення ✅
- **Технологічні карти** (налаштування в «Manufacturing Plans Settings») ✅

- Модуль «Виробництво у Проектах»: вкладки Специфікація + Товари у проекті + Виробництво ✅

- Немає маршрутів/маршрутних листів

**Для кого:** Малий/середній custom-order виробник (меблі, подарунки, контрактне); CRM-first компанія з базовими потребами виробничого обліку

## ⚠️ Підводні камені

### 1. BAS ≠ безпечна заміна 1С
BAS — це **1С-код з польським power of attorney**. Технічно ідентична архітектура. BAS ERP, BAS Бухгалтерія КОРП, BAS Документообіг КОРП — у переліку ДССЗЗІ від 09.01.2026. Будь-яке впровадження на BAS = той самий ризик, що і 1С.

### 2. Швейка 8 та BAS Харчове виробництво — популярні, але під забороною
Тисячі українських виробників використовують ці рішення. Вони ЗРУЧНІ та ФУНКЦІОНАЛЬНІ. Але обидва базуються на BAS-платформі = під санкціями. Строк поетапної відмови — до 2030 р., але для КІІ — вже зараз.

### 3. Базис-Мебельщик та Julivi — підводні камені для меблевиків та швейників
- **Базис-Мебельщик** — домінуюче CAD-рішення для корпусних меблів в UA — РОСІЙСЬКИЙ продукт (ООО «Базис-Центр», Коломна, bazissoft.ru). Не в переліку ДССЗЗІ поіменно, але однаковий ризик. Впровадженці часто «не згадують» про це. Безпечні альтернативи: Астра Конструктор Меблів (UA), PRO100 (Польща).
- **Julivi** (julivi.ru) — популярне CAD-рішення для швейного виробництва — РОСІЙСЬКИЙ продукт (ТОВ «САПРЛЕГПРОМ», Москва). Безпечна альтернатива: **Грація** (graziacad.in.ua, Україна ✅), міжнародні: Gerber AccuMark, Lectra Modaris.

### 4. Odoo Community vs Enterprise — підводний камінь
Community = безкоштовна ліцензія, але: немає офіційної підтримки SAP, немає Odoo Studio, немає деяких enterprise-модулів (Forecasting, MPS, IoT). Багато хто стартує на Community і через рік змушений перейти на Enterprise через відсутню функціональність.

### 5. Perfectum — виробництво прив'язане до Проектів
Якщо ваш бізнес — не «проектний», а серійний або потоковий, Perfectum виробничий модуль потребуватиме кастомізації. Підходить для custom-order, не для масового виробництва.

### 6. Dynamics 365 BC — Essential ≠ виробництво
$80/user/міс — ТІЛЬКИ Essential (фінанси, склад, CRM). Виробничий модуль = ТІЛЬКИ Premium ($110/user). Типова помилка: клієнт бачить $80, купує Essential, потім дізнається що виробництво — у Premium.

### 7. Zoho — ні в якому разі не Zoho ERP для UA
Zoho ERP (з повним виробничим модулем) запущений ЛИШЕ В ІНДІЇ (23.01.2026). Для України — недоступний на 2026. Zoho Inventory (Composite Items = BOM) — лише для простого складального виробництва. Немає MRP, немає маршрутів. Немає UA-інтерфейсу.

### 8. Впровадження ERP — тривале і коштовне
- Odoo Community: 50–200 годин впровадження → від 30K UAH (стартовий) до 300K UAH (повний виробничий проект)
- SAP B1: $30K–80K
- Dynamics 365 BC: $30K–100K
- IT-Enterprise: 6–24 місяці
Без бюджету на впровадження — краще Perfectum або KeyCRM+Bimp

### 9. KeyCRM та KeepinCRM — НЕ для серійного виробництва
Їх «виробничий модуль» — це трекінг статусів виробничих запитів, а НЕ BOM + MRP + собівартість. Підходить для мікро-workshop (замовлення ≤ 100/місяць, 1–3 майстри), не для реального виробника.

## 📋 Чек-лист вибору системи

**Для виробника будь-якої ніші:**
- [ ] Потрібен повноцінний BOM + MRP + собівартість + маршрути? → **Odoo MRP** (SMB) або **SAP B1** (mid+)
- [ ] Великий промисловий завод, MES, APS? → **IT-Enterprise**
- [ ] Малий custom-order бізнес (меблі, подарунки, кастомні товари)? → **Perfectum CRM+ERP** (найдешевший старт)
- [ ] Швейне/одяг? → **Bimp** (нативно для текстилю) або **Odoo** (якщо ширший функціонал)
- [ ] Харчове виробництво? → **Odoo** (LOT, рецептури, byproduct) або **SAP B1 + Beas**
- [ ] Контрактне B2B (метал, пластик, друк)? → **Odoo** (CRM+виробництво в одному)
- [ ] Товарка + маркетплейси (Prom/Rozetka) + просте виробництво? → **KeyCRM** або **Perfectum**
- [ ] Хочу Microsoft-екосистему? → **Dynamics 365 BC Premium** ($110/user)
- [ ] Є бюджет на ERP-впровадження $30K+? → **SAP B1** або **Dynamics 365 BC**
- [ ] Немає бюджету, потрібен старт? → **Odoo Community** (ліцензія безкоштовна) + UA-партнер
