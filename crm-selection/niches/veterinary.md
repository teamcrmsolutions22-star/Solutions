## Ключовий висновок ринку

Ветеринарний бізнес в Україні — унікальна ніша: **пацієнт = тварина, платник = власник** (два окремих записи). Система повинна розуміти цю подвійну архітектуру нативно, а не через «хак» з кастомними полями.

Ринок ділиться на два класи:
- **Спеціалізовані ветеринарні PIMS** (Practice Information Management System): ENOTE, Jet Vet, Clinica Web — побудовані саме «для ветів»; картка тварини + власник + медкартка + аптека в ядрі системи
- **Адаптовані booking/CRM**: Appointer, EasyWeek, KeyCRM — покривають запис і базову картку, але медична логіка (анамнез, призначення, списання препаратів) вимагає кастомізації

**Критичні попередження:**
- **Vetmanager** 🚫🇷🇺 — компанія зареєстрована у Краснодарі (Росія); незважаючи на сайт vetmanager.com.ua — НЕ рекомендується
- **Altegio** 🚫🇷🇺 — популярний для грумінгу/зоосалонів, але російська система (санкції РНБО)
- **YCLIENTS** 🚫🇷🇺 — те саме
- **Bitrix24** 🚫🇷🇺, **1С:Ветклініка** 🚫🇷🇺 — повністю санкціоновані

Міжнародні PIMS (Digitail, Provet Cloud, ezyVet) — **технічно доступні як SaaS, але без UA-локалізації**, без ПРРО, без інтеграції з Єдиним реєстром домашніх тварин. Підходять для клінік з міжнародним профілем або власниками-іноземцями.

## Топ-3 рекомендації по сегментах

### A. Ветклініка (прийоми, стаціонар, аптека)

**Ключові вимоги:** повна медкартка, списання препаратів, аптечний облік, ПРРО, нагадування про вакцинацію, аналітика лікарів.

1. **ENOTE** — найбільш повне українське рішення з нативною аптекою (і ветаптекою), інтеграцією ПРРО Checkbox, повною медкарткою, лабораторними результатами та call-центром з ідентифікацією номера. Єдина UA-система, яка покриває «клініку + аптеку + зоомагазин» в одному продукті.
2. **Jet Vet** — зрілий UA-продукт (5+ років, 500+ клінік), модуль аптеки та препаратів, JetVetPass для власників тварин (iOS/Android), шаблони призначень «1 клік». Від 250 грн/user/міс — оптимальне співвідношення ціни та функціоналу.
3. **Clinica Web** — якщо потрібна MIS-орієнтована система з акцентом на медичний документообіг, чергу та контроль кількох кабінетів. Ціна уточнювати.

---

### B. Веткабінет / приватний ветеринар (малий формат)

**Ключові вимоги:** простота, мінімальна ціна, запис, картка тварини, нагадування.

1. **Jet Vet (мінімальний тариф, 250 грн/user)** — 14 днів безкоштовно, одразу можна почати вести прийоми, JetVetPass для клієнтів включений.
2. **Appointer** — якщо вже знайомий з booking-логікою (як для beauty); картки пацієнтів, авто-SMS, Checkbox. Мінімальна ціна.
3. **EasyWeek (free tier)** — для зовсім маленького кабінету без аптеки; основне — онлайн-запис і нагадування. При зростанні переходити на спеціалізовану систему.

---

### C. Мережа ветклінік (мультилокація)

**Ключові вимоги:** централізована база тварин/власників, консолідована аналітика, управління персоналом, мультилокація.

1. **ENOTE** — виділені серверні тарифи для мереж, можливість об'єднати кілька локацій, єдина база клієнтів + аптека + фінанси.
2. **Jet Vet** — підтримка мережі клінік, централізована картотека, права доступу за клінікою/лікарем.
3. **Digitail** (для міжнар. мереж або клінік з іноземними власниками) — AI-native, Pet Parent App, 30+ інтеграцій, відмінна аналітика мережі. Але: без UA-локалізації та ПРРО; $300/DVM/міс.

---

### D. Зооготель / грумінг / зоосалон

**Ключові вимоги:** booking (як Beauty), картка тварини+власник, абонементи, нагадування, ПРРО.

1. **EasyWeek** — еталонна booking-логіка для грумінгу/зооготелю: онлайн-запис 24/7, розклад майстрів/кабінетів, нагадування. Реальні кейси грумінг-салонів на EasyWeek.
2. **Appointer** — аналог EasyWeek, але з Checkbox (ПРРО) нативно + картки клієнтів з нотатками про тварину. Підходить для невеликих грумінгів/зооготелів.
3. **CleverBOX** — якщо поєднуєте грумінг + косметологія + ветпослуги; клієнтська база з историею, програми лояльності.

⚠️ **Altegio** 🚫🇷🇺 та **YCLIENTS** 🚫🇷🇺 — популярні для грумінгу, але санкціоноване ПЗ. Замінники: EasyWeek, Appointer, CleverBOX.

### E. Зоомагазин + ветпослуги (роздріб + послуги)

**Ключові вимоги:** товарний облік (корми/аксесуари) + послуги (консультації/щеплення) + ПРРО + картка тварини.

1. **ENOTE** — єдиний UA-продукт, де поєднано ветклініку + ветаптеку + зоомагазин в одній системі. Облік товарів, ПРРО, медкартки — все разом.
2. **KeyCRM** — якщо магазин активно продає через інтернет/маркетплейси (Rozetka, Prom, Instagram) та хоче додати ветпослуги; $19/акаунт flat, омніканал. Медкартка — через кастомні поля.
3. **Appointer + окремий товарний облік** — для невеликих зоомагазинів з кількома ветпослугами: Appointer для запису/картки, окрема товарна система або ПРРО-рішення для роздрібу.

## Блок: картка «тварина + власник» + медкартка

### Архітектурна різниця — критично

> У ветеринарії існують **дві пов'язані картки**: картка власника (контакти, оплата) і картка тварини (вид/порода/вік/вага/чіп). На одного власника може бути кілька тварин. Це докорінно відрізняється від медицини або beauty, де пацієнт і платник — одна особа.

| Рівень підтримки | Системи |
|-----------------|---------|
| ✅ Нативна подвійна архітектура | ENOTE, Jet Vet, Clinica Web, Digitail, Provet Cloud, ezyVet |
| ± Через кастомні поля / зв'язки | Appointer, KeyCRM (картка тварини як додаткові поля до клієнта) |
| ❌ Не підтримується нативно | EasyWeek, KeepinCRM (треба обходити) |

### Що входить у повноцінну медкартку (ENOTE, Jet Vet)
- Вид/порода/вік/вага/стать/чіп/мітка
- Анамнез та хронічні хвороби
- Алергії та протипоказання
- Щеплення + дегельмінтизація (графік, нагадування)
- Діагнози, призначення, курси лікування
- Результати лабораторних аналізів і досліджень
- Фото/відео тварини
- Рецепти та виписки
- Оцифрований ветпаспорт (у JetVetPass — мобільно)

### Мобільна картка власника (JetVetPass)

Jet Vet має унікальну фічу: **JetVetPass** — безкоштовний мобільний додаток для власників тварин (iOS + Android), де видно:
- Повну медичну історію (Premium — необмежено, Free — 3 останніх візити/30 днів)
- Заплановані прийоми
- Нагадування від клініки (push безкоштовно)
- Запис на прийом
- Необмежена кількість тварин (Premium)

## Блок: нагадування про вакцинацію / повторні візити (retention)

Нагадування — **головний драйвер повторних візитів** у ветеринарії. Власники часто не пам'ятають про дегельмінтизацію кожні 3 міс або щеплення раз на рік.

### Рівень підтримки нагадувань

| Рівень | Системи | Деталі |
|--------|---------|--------|
| ✅ Автоматичні за графіком | ENOTE, Jet Vet, Appointer | Адміністратор задає розклад → система автоматично відправляє |
| ✅ Booking + нагадування | EasyWeek, CleverBOX | Підтвердження запису + нагадування за 24 год |
| ± Ручне або через тригери | KeyCRM | Можна налаштувати тригери, але не нативно ветеринарний розклад |
| ✅ Авто + Push (клієнтський додаток) | Jet Vet (JetVetPass) | Push у додатку — безкоштовно, не SMS |
| ✅ Авто + Email/SMS/Push | Digitail, Provet Cloud | Але тільки англійські шаблони |

### Канали нагадувань (UA-системи)
- **SMS** — Appointer, Jet Vet (через SMS-gateway)
- **Email** — ENOTE, Clinica Web, Jet Vet
- **Push у мобільному** — Jet Vet (JetVetPass) — найдешевший канал
- **Telegram/Viber** — KeyCRM (нативно); інші — ⚠️ не підтверджено публічно для більшості vet-систем

## Блок: аптека/склад + роздріб кормів + ПРРО

### Аптечний облік — специфіка ветеринарії

Ветаптека поєднує:
- Приходування препаратів (постачальники, партії, терміни придатності)
- Списання на прийом (автоматичне при збереженні призначення)
- Продаж через касу (кормів, аксесуарів, препаратів)
- ПРРО для фіскалізації

| Система | Аптека/склад | Списання | Роздріб | ПРРО | Примітки |

### ПРРО для ветеринарії — важливо

Ветеринарна клініка = платник єдиного податку або загальна система → **ПРРО обов'язковий** з 01.01.2024 для всіх, хто приймає оплату готівкою/карткою.

- **Checkbox** — найпоширеніший UA ПРРО; підтримує ENOTE та Appointer
- Ціна Checkbox: від 249 грн/міс (станом на 2025)
- ENOTE: інтеграція через Desktop-додаток з offline-режимом (на випадок відключення інтернету)

## AI-функції в ветеринарному бізнесі (2025–2026)

### Що реально працює в UA-системах

**ENOTE, Jet Vet, Clinica Web:** публічно не заявляють власних AI-модулів станом на 2026. Шаблони призначень «1 клік» у Jet Vet — найближче до автоматизації, але не AI.

**KeyCRM:** можлива інтеграція з ChatGPT/Claude через API для генерації повідомлень клієнтам.

### Що реально працює в міжнародних системах

**Digitail — флагман AI серед vet PIMS (2025–2026):**
- 15+ вбудованих AI workflows
- **AI SOAP Notes** — автоматичне створення медичних записів під час прийому (генерація тексту по голосовому вводу або структурованих даних)
- Discharge instructions — автогенерація виписки для власника
- Patient history summaries — стислий огляд медісторії тварини
- Уточнення: AI-функції можуть бути як add-on → перевірити на digitail.com/plans

**Provet Cloud — AI SOAP:**
- Вбудований AI-асистент для SOAP-нотаток під час консультації
- Оцінюється на 45+ країн, EN-орієнтований

**Vetstoria (booking add-on):**
- Аналітика бронювань з AI-рекомендаціями
- Не PIMS — потребує інтеграції з основною системою

### Практичний висновок для UA-ринку

> AI у ветеринарії 2025–2026 — реально лише у міжнародних системах (Digitail, Provet). UA-системи (ENOTE, Jet Vet) поки зосереджені на автоматизації процесів (шаблони, нагадування), а не генеративному AI. Якщо AI у SOAP-нотатках — пріоритет → Digitail (EN), якщо UA-ринок і ПРРО → ENOTE або Jet Vet.

## Підводні камені та застереження

### 1. Vetmanager 🚫🇷🇺 — популярний, але Росія

Vetmanager має сайт vetmanager.com.ua і активно використовувався в Україні. Проте компанія зареєстрована у **Краснодарі, Росія**. Для клінік, що розглядають переїзд або досі використовують — **критично мігрувати** на ENOTE, Jet Vet або Clinica Web. Ризики: той самий, що й з Bitrix24 — юрисдикція РФ, доступ держорганів РФ до баз клієнтів.

### 2. Altegio 🚫🇷🇺 та YCLIENTS 🚫🇷🇺 для грумінгу

Обидві системи дуже поширені серед грумінгів і зоосалонів через зручний booking-інтерфейс. Обидві — **санкціоновані РФ-системи**. Альтернативи: **EasyWeek**, **Appointer**, **CleverBOX** — покривають той самий функціонал онлайн-запису без ризику.

### 3. Пастка «beauty CRM для ветів»

EasyWeek, Appointer, CleverBOX — відмінні для запису і нагадувань, але **не мають аптечного обліку та повноцінної медкартки**. Ветклініка, що побудує основний workflow на booking-системі, не зможе відстежувати списання препаратів, вести SOAP-нотатки і генерувати звіти billable per vet. При зростанні клініки доведеться мігрувати → закладати бюджет на перехід.

### 4. Міжнародні PIMS — відсутність UA-специфіки

Digitail ($300/DVM), Provet Cloud (€25/user), ezyVet ($245+) — відмінні системи, але:
- ❌ Немає українського інтерфейсу
- ❌ Немає ПРРО (доведеться підключати окрему касу)
- ❌ Немає інтеграції з Єдиним реєстром домашніх тварин (vet.pet.gov.ua)
- ❌ Ціна у 10–30x вища, ніж UA-аналоги
- ✅ Мають сенс для клінік з іноземними власниками, міжнародними стандартами або EN-практикою

### 5. Єдиний державний реєстр домашніх тварин (vet.pet.gov.ua)

З 2024 року ветлікарі зобов'язані вносити дані про вакцинацію у державний реєстр. **Жодна комерційна CRM поки не має нативної безшовної інтеграції** з vet.pet.gov.ua «з коробки». Vetmanager.com.ua рекламував цю функцію, але система — РФ, тому не рекомендується. Для UA-систем — уточнювати у ENOTE та Jet Vet на демо.

### 6. Складність порівняно з beauty/медициною

Ветеринарний бізнес — складніший за beauty (є аптека + медкартка) і простіший за медицину (немає eHealth/НСЗУ/КСЗІ). Не варто вибирати «медичну МІС» з людської медицини (nHealth, Medics) — вони не мають картки тварини та аптечного vet-обліку.
