## Ключовий висновок ринку

Ветеринарний бізнес в Україні — унікальна ніша: **пацієнт = тварина, платник = власник** (два окремих записи, один власник може мати кілька тварин). Система повинна розуміти цю подвійну архітектуру нативно, а не через «хак» з кастомними полями.

Ринок ділиться на два класи:
- **Спеціалізовані ветеринарні PIMS** (Practice Information Management System): FerretID, Jet Vet, Clinica Web — побудовані саме «для ветів»; картка тварини + власник + медкартка + аптека в ядрі системи
- **Адаптовані booking/CRM**: Appointer, EasyWeek, RO App, KeyCRM — покривають запис і базову картку, але медична логіка (анамнез, призначення, списання препаратів) вимагає кастомізації або зв'язки з PIMS

> **Ринковий контекст (2025):** ⚠️ За даними delo.ua (2025), українці витратили понад 200 млн грн на здоров'я домашніх тварин — цей ринок демонструє стабільне зростання навіть в умовах війни, що стимулює розвиток UA-систем управління ветбізнесом.

**Критичні попередження:**
- **Vetmanager** 🚫🇷🇺 — компанія зареєстрована у Краснодарі (Росія), внесена до Єдиного реєстру рос. ПЗ, сервери в РФ/Білорусі (і Німеччині для міжнар. клієнтів); у документації: *«Як акредитована IT-компанія, гарантуємо відповідність усім вимогам **російського законодавства** (152-ФЗ, 54-ФЗ)»* — НЕ рекомендується
- **ENOTE** ⚠️ **подвійний домен**: enote.vet = ТОВ "Програмні системи плюс" (Житомир, Україна, з 2007 р., 680 грн/user/міс у гривнях, є в переліку vet.pet.gov.ua для інтеграції з держреєстром; включає ERP-рівень: P&L, рух коштів, стаціонар, аптека, інтеграція з ПРРО Cashalot); **enote.cloud** = окрема платформа з RUB-цінами (37 930 RUB/міс за 20 users), орієнтована на РФ-ринок — **використовувати ЛИШЕ enote.vet, уточнювати юрисдикцію серверів на демо**
- **Altegio** 🚫🇷🇺 — популярний для грумінгу/зоосалонів, але Russian-origin: засновник Юрій Петров → Altegio Limited (Кіпр) → Altegio Ukraine LLC (суб-ліцензіат, ЄДРПОУ 44662266, зареєстрована квітень 2022 на Моніку Ковальську); App Store copyright: Altegio Limited (рос. капітал). Незважаючи на офіційну позицію "Stand with Ukraine" — юридичний зв'язок з РФ підтверджений медіа-розслідуваннями (argumentua.com)
- **YCLIENTS** 🚫🇷🇺 — 100% VK (Росія), викуплений у 2024
- **Bitrix24** 🚫🇷🇺, **1С:Ветклініка** 🚫🇷🇺, **БИТ:Айболит** 🚫🇷🇺 — повністю санкціоновані, HQ Калінінград

Міжнародні PIMS (Digitail, Provet Cloud, ezyVet) — **технічно доступні як SaaS, але без UA-локалізації**, без ПРРО, без інтеграції з Єдиним реєстром домашніх тварин. Підходять для клінік з міжнародним профілем або власниками-іноземцями. Також існує **Shepherd** (US PIMS) — заявляє "Smart AI tools" для автоматизації медичних записів та інвойсів, але UA-доступність і локалізація не підтверджені — ⚠️ не розглядати для UA-ринку без перевірки.

## Блок: картка «тварина + власник» + медкартка

### Архітектурна різниця — критично

> У ветеринарії існують **дві пов'язані картки**: картка власника (контакти, оплата, юридичні згоди) і картка тварини (вид/порода/вік/вага/чіп). На одного власника — кілька тварин. Потрібна підтримка передачі права власності зі збереженням медісторії тварини.

| Рівень підтримки | Системи |
|-----------------|---------|
| ✅ Нативна подвійна архітектура | FerretID, Jet Vet, Clinica Web, Digitail, Provet Cloud, ezyVet |
| ✅ Нативна подвійна архітектура (UA) | ENOTE (enote.vet) — власник + кілька тварин, анамнез, вакцинації |
| ± Через кастомні поля / шаблони | Appointer (картка тварини як кастомний шаблон), RO App (RemOnline) |
| ⚠️ Трекер медданих (нотатки без структури) | EasyWeek — "зберігати всю медичну історію" але без SOAP/діагнозів |
| ❌ Не підтримується нативно | KeyCRM (тварина = кастомні поля клієнта), KeepinCRM |

> ⚠️ **Пастка:** KeyCRM / RO App / EasyWeek з кастомними полями задовольняють грумінг або зоомагазин, але в повноцінних клініках — хаос у медкартках: відсутність SOAP, діагнозів, анамнезу. Найбільш стабільна архітектура для клініки: **PIMS (FerretID / Clinica Web / Jet Vet) + CRM (KeyCRM / KeepinCRM) + ПРРО (Checkbox)**.

### Що входить у повноцінну медкартку (FerretID, Jet Vet, Clinica Web)
- Вид/порода/вік/вага/стать/чіп/мітка
- Анамнез та хронічні хвороби
- Алергії та протипоказання
- Щеплення + дегельмінтизація (графік, нагадування про ревакцинацію)
- Діагнози, призначення, курси лікування
- Результати лабораторних аналізів і досліджень
- Фото/відео тварини; рентгени (Clinica Web)
- Рецепти та виписки для власника

> **Рівень «ідеальної» ЕМК** (переважно міжнародні PIMS — ezyVet, Provet, Digitail): шаблони SOAP (Суб'єктивно/Об'єктивно/Аналіз/План), інтеграція з лабораторними аналізаторами **IDEXX** і **Zoetis**, цифрове зберігання рентгенів і МРТ у форматі **DICOM**. Більшість UA-систем станом на 2025–2026 покривають ЕМК на рівні структурованих текстових полів + прикріплення файлів (PDF/JPG). Це стимулює локальних розробників до швидкого розвитку медичних модулів.

### Мобільна картка власника (JetVetPass)

Jet Vet має унікальну фічу: **JetVetPass** — безкоштовний мобільний додаток для власників тварин (iOS + Android), де видно:
- Повну медичну історію (Premium — необмежено, Free — 3 останніх візити/30 днів)
- Заплановані прийоми та нагадування від клініки (push-повідомлення — безкоштовно)
- Запис на прийом онлайн
- Необмежена кількість тварин (Premium)
- Оцифрований ветпаспорт

## Блок: нагадування про вакцинацію / повторні візити (retention)

Нагадування — **головний драйвер повторних візитів** у ветеринарії. Власники часто не пам'ятають про дегельмінтизацію кожні 3 міс або щеплення раз на рік. Автоматизація підвищує Retention rate до 70%+ проти 40–50% при ручному обзвоні.

### Рівень підтримки нагадувань

| Рівень | Системи | Деталі |
|--------|---------|--------|
| ✅ Нативний ветеринарний календар | FerretID | Окремий модуль «Вакцинації», звіт/export, нагадування про ревакцинацію за видом вакцини |
| ✅ Нагадування за розкладом | Jet Vet, Appointer | Апоінтер: через 11 міс після дати щеплення → авто-SMS |
| ✅ Авто + push (JetVetPass) | Jet Vet | Push у застосунку власника — найдешевший канал |
| ✅ Нагадування про планові процедури | EasyWeek | Автоматичні сповіщення на основі дати попереднього візиту |
| ✅ Нагадування через месенджери | CleverBOX, KeyCRM | Viber/Telegram/SMS; KeyCRM — тригери «через N місяців» |
| ⚠️ Авто + Push (EN-only, без UA-ПРРО) | Digitail, Provet Cloud | Push у Pet Parent App + SMS/Email |

### Канали нагадувань (UA-системи)
- **SMS** — Appointer, Jet Vet, RO App (через SMS-gateway)
- **Email** — FerretID, Clinica Web, Jet Vet
- **Push у мобільному** — Jet Vet (JetVetPass) — безкоштовно
- **Telegram/Viber** — CleverBOX (нативно), KeyCRM (нативно), Appointer (TG вбудований)

### Практичний шаблон retention-архітектури

```
PIMS (FerretID / Clinica Web / Jet Vet)
  → зберігає дати вакцинацій, призначень
  → генерує список майбутніх щеплень
     ↓ (щоденний/тижневий export)
CRM / месенджер (KeyCRM / KeepinCRM / Appointer / CleverBOX)
  → відправляє Viber/Telegram/SMS-нагадування власнику
```

## AI-функції в ветеринарному бізнесі (2025–2026)

### Що реально працює в UA-системах (2025–2026)

**FerretID, Jet Vet, Clinica Web, Appointer, EasyWeek, RO App:** публічно не заявляють власних ML/AI-модулів станом на 2026. Автоматизація — на рівні тригерів і шаблонів (шаблони призначень «1 клік» у Jet Vet — найближче до автоматизації, але не генеративний AI).

**KeyCRM / RO App (KeepinCRM):** AI-асистенти для обробки вхідних повідомлень у чатах (чернетки відповідей, контекстний аналіз) — у розробці/бета станом на 2025.

### Що реально працює в міжнародних системах

**Digitail — флагман AI серед vet PIMS (2025–2026):**
- 15+ вбудованих AI workflows
- **AI SOAP Dictation** — диктуєте нотатки під час прийому → система генерує структурований SOAP-запис
- AI discharge instructions — автогенерація виписки для власника
- AI patient history summaries — стислий огляд медісторії тварини перед прийомом
- AI invoice automation — автоматична прив'язка процедур до рахунку
- AI practice manager — клінічні рекомендації, адміністрування
- Уточнення: частина AI — як add-on → перевіряти на digitail.com/plans

**Provet Cloud:**
- **AI Scribe** — автоматичне документування консультації в реальному часі
- AI Assistant для нотаток і резюме візиту

### Практичний висновок для UA-ринку

> AI у ветеринарії 2025–2026 — реально лише у міжнародних системах (Digitail, Provet). UA-системи зосереджені на автоматизації процесів (шаблони, нагадування), а не генеративному AI.
> - Якщо AI SOAP-нотатки + Pet Parent App → **Digitail** ($300/DVM/міс, EN)
> - Якщо UA-ринок, ПРРО, підтримка українською → **FerretID** або **Jet Vet**

### Voice-to-Text (голос → медкартка) — новий тренд 2025–2026

Нова практика, що набирає популярності у ветклініках: лікар диктує анамнез або протокол операції безпосередньо у смартфон → AI конвертує аудіо у структурований текст медкартки. Це заощаджує **10–20 хвилин на оформленні кожного пацієнта**.

- **Digitail** — AI SOAP Dictation є нативною функцією (диктування під час прийому → готовий SOAP-запис)
- **Provet Cloud** — AI Scribe (документування консультації у реальному часі)
- **UA-системи (FerretID, Clinica Web, Appointer, EasyWeek)** — станом на 2026 реалізовано переважно через сторонні інтеграції iOS/Android (системні функції розпізнавання мови), а не нативний AI-модуль

### Предиктивна аналітика та LTV

Концепція LTV (Lifetime Value — пожиттєва цінність клієнта) у ветеринарії формується насамперед за рахунок **регулярних профілактичних заходів**, а не екстреної хірургії: щорічна комплексна вакцинація, щоквартальна дегельмінтизація, сезонна обробка від кліщів та бліх.

- **RO App** — 20+ типів звітів дозволяє відстежувати сезонні тренди (сплески бабезіозу навесні, гастроентерити в святкові дні) для оптимізації закупівель і графіків персоналу
- **EasyWeek** — аналітика завантаженості, ефективності реклами
- **KeyCRM** — потужна e-commerce аналітика, воронка, P&L

> Автоматизація нагадувань підвищує Retention rate до **70%+** проти 40–50% при ручному обзвоні — і є головним способом збільшення LTV у ветеринарії.

## Мобільність та безпека даних

### Мобільний доступ

Сучасна ветеринарна практика виходить за межі кабінету: виклики додому, обходи стаціонару, велика за площею клініка. Системи з повноцінним мобільним застосунком:

| Система | Мобільний для лікаря/адміна | Мобільний для клієнта | Примітки |
|---------|----------------------------|-----------------------|---------|
| **Jet Vet** | ✅ | ✅ JetVetPass (iOS/Android) | Пуш-нагадування у JetVetPass — безкоштовно |
| **Appointer** | ✅ 120 грн/міс | ✅ | QR-оплата ПриватБанк/Монобанк без POS-термінала |
| **EasyWeek** | ✅ (окремий app для бізнесу) | ✅ (окремий app для клієнтів) | Два різних застосунки |
| **KeyCRM** | ✅ | — | Омніканал у мобільному |
| **RO App** | ✅ | — | Включено у всі тарифи |
| **Digitail** | ✅ | ✅ Pet Parent App (iOS/Android) | AI SOAP dictation з мобільного |

### Безпека та юрисдикція серверів

Провідні UA-платформи (Appointer, RO App, KeyCRM) зберігають дані на захищених серверах в Україні або ЄС, дотримуючись Закону України «Про захист персональних даних». EasyWeek — відповідність стандартам PCI DSS, SSL, GDPR. ⚠️ При виборі будь-якої системи — обов'язково уточнювати юрисдикцію серверів зберігання персональних даних клієнтів (телефони, платіжна інформація, адреси).

> **Критичний ризик рос. ПЗ:** Витік персональних даних українських власників тварин (бази телефонів, геолокації клінік, платіжна інформація) до структур ФСБ є реальним ризиком при використанні ПЗ з рос. юрисдикцією. Vetmanager, ENOTE (enote.cloud), Altegio — підпадають під цю категорію ризику.

## Підводні камені та застереження

### 1. Vetmanager 🚫🇷🇺 — поширений, але Росія

Vetmanager мав сайт vetmanager.com.ua і активно використовувався в Україні. Але компанія зареєстрована у **Краснодарі, Росія**, офіційно внесена до Єдиного реєстру рос. ПЗ. Сервери розташовані в РФ/Білорусі (для рос. клієнтів) і Німеччині (для міжнародних). У офіційній документації компанії: *«Як акредитована IT-компанія, гарантуємо відповідність усім вимогам **російського законодавства** (152-ФЗ, 54-ФЗ)»*. Ризики: юрисдикція РФ, передача клієнтських даних, ДССЗЗІ-ризики, санкційний ризик. Мігрувати на FerretID, Jet Vet або Clinica Web.

### 2. ENOTE — увага на правильний домен ⚠️

ENOTE існує у двох версіях:
- **enote.vet** — продукт ТОВ "Програмні системи плюс" (Житомир, Україна, з 2007 р.); 680 грн/user/міс у гривнях; включений у перелік систем для інтеграції з держреєстром домашніх тварин vet.pet.gov.ua. **Це легітимна українська система.**
- **enote.cloud** — окрема платформа з RUB-цінами (37 930 RUB/міс за 20 users), орієнтована на рос. ринок. **НЕ використовувати.**

**Практична порада:** При виборі завжди уточнюйте — ви підключаєтесь до enote.vet (UA) чи enote.cloud (RU). Перевіряйте юрисдикцію серверів зберігання даних на демо. Різниця між двома доменами — принципова.

### 3. Altegio 🚫🇷🇺 та YCLIENTS 🚫🇷🇺 для грумінгу — деталі

Altegio заявляє незалежність від YCLIENTS (ребрендинг після скандалу) та декларує позицію "Stand with Ukraine", офіційно не працює з РФ/РБ. Але незалежні медіа-розслідування (argumentua.com) фіксують: Юрій Петров (засновник YCLIENTS) → Altegio Limited (Кіпр) → Altegio Ukraine LLC (суб-ліцензіат, **ЄДРПОУ 44662266**, зареєстрована квітень 2022 на Моніку Ковальську). В App Store copyright: **Altegio Limited** (Кіпр, рос. капітал). Ліцензійний договір: українська компанія — лише субліцензіат кіпрської структури. **Аргумент для клієнта:** *«Персональні дані ваших клієнтів — телефони, адреси, фінансові транзакції — зберігаються у системі, що корінням пов'язана з РФ через засновника YCLIENTS, купленого VK (структура, підконтрольна близькому оточенню Путіна)».*

### 4. Пастка «beauty CRM для ветів»

EasyWeek, Appointer, CleverBOX — відмінні для запису і нагадувань, але **не мають повноцінної медкартки та аптечного обліку** у ветеринарному розумінні. Ветклініка, що побудує основний workflow на booking-системі, не зможе відстежувати SOAP-нотатки, списання препаратів, звіти billable per vet. При зростанні клініки доведеться мігрувати.

### 5. Міжнародні PIMS — відсутність UA-специфіки

Digitail ($300/DVM), Provet Cloud (€25/user), ezyVet ($245+) — відмінні системи, але:
- ❌ Немає українського інтерфейсу
- ❌ Немає ПРРО (потрібна окрема каса)
- ❌ Немає інтеграції з Єдиним реєстром домашніх тварин (vet.pet.gov.ua)
- ❌ Ціна у 10–30x вища, ніж UA-аналоги
- ✅ Мають сенс для клінік з іноземними власниками, EN-практикою або преміум-сегменту

### 6. Єдиний державний реєстр домашніх тварин (vet.pet.gov.ua)

З 2024 ветлікарі зобов'язані вносити дані про вакцинацію у державний реєстр (vet.pet.gov.ua). Реєстр інтегровано з **Diia.Engine** — тобто дані про щеплення мають потенційну прив'язку до державної цифрової інфраструктури. На сайті vet.pet.gov.ua є перелік інтегрованих систем — **підтверджено наявність Jet Vet і ENOTE (enote.vet)**. FerretID і Clinica Web — уточнювати наявність API-інтеграції на демо. Більшість booking-систем (Appointer, EasyWeek, CleverBOX) — **не інтегровані з держреєстром** (2025–2026).

> ⚠️ **Правовий ризик:** Використання рос. ПЗ (Vetmanager, ENOTE enote.cloud) для передачі даних у держреєстр vet.pet.gov.ua, пов'язаний з Diia.Engine — становить потенційний ризик витоку персональних даних громадян до юрисдикції РФ.

### 7. ENOTE (enote.vet) — деталі функціоналу та ціноутворення

ENOTE (enote.vet) — функціонально найближча до повного ERP серед UA-специфічних рішень:
- **Клінічний рівень:** SOAP, аналізи, стаціонар, модуль препаратів і аптека
- **Фінансовий рівень:** P&L (звіт про прибутки та збитки), рух коштів (cashflow), баланс
- **ПРРО:** інтеграція з Cashalot (інструкції API-COM на cashalot.ua — підтверджено)
- **Ціна (enote.vet UA):** 680 грн/user/міс
- **Ціна (enote.cloud RU — НЕ використовувати):** 37 930 RUB/міс за 20 users

При виборі: **завжди уточнювати домен (enote.vet vs enote.cloud) і юрисдикцію серверів зберігання даних** — це принципово різні продукти.

## Швидкий довідник вибору

```
Повна ветклініка (прийоми + стаціонар + аптека)?        → FerretID або Clinica Web
ERP-рівень (P&L, cashflow, аптека, ПРРО, UA)?           → ENOTE (enote.vet) ⚠️ уточнити сервери
UA-система + мобільний додаток для клієнтів?            → Jet Vet (JetVetPass)
Малий кабінет / старт / мінімальний бюджет?             → Jet Vet (250 грн) або EasyWeek (Free)
Грумінг / зоосалон / зооготель?                         → EasyWeek або Appointer (НЕ Altegio 🚫🇷🇺)
Зоомагазин + ветпослуги (omnichannel)?                  → KeyCRM + Jet Vet/FerretID
Мережа клінік з UA-підтримкою?                          → Clinica Web або FerretID
AI SOAP notes + Pet Parent App (EN-орієнтація)?         → Digitail ($300/DVM)
Voice-to-Text нотатки (EN)?                             → Digitail (AI SOAP Dictation) або Provet Cloud (AI Scribe)
Інтеграція з держреєстром тварин (vet.pet.gov.ua)?      → Jet Vet або ENOTE (enote.vet) — підтверджено
Використовуєте Vetmanager?                               → Мігрувати 🚫🇷🇺 → FerretID або Clinica Web або ENOTE (enote.vet)
Використовуєте enote.cloud (RUB-версію)?                 → Перейти на enote.vet (UA) або FerretID
Використовуєте Altegio або YCLIENTS?                    → Мігрувати 🚫🇷🇺 → EasyWeek або Appointer
```

---

*Огляд відображає публічно доступну інформацію станом на червень 2026. Перед вибором — обов'язково демо та уточнення: FerretID (ferretid.com), Jet Vet (jet.vet), Clinica Web (clinica-web.ua), RO App (roapp.com.ua), ENOTE (enote.vet). Особливо перевірити: ПРРО у FerretID і Jet Vet, інтеграція з держреєстром тварин (vet.pet.gov.ua), юрисдикція серверів у ENOTE. Ціни KeyCRM — уточнити модель (per volume) на help.keycrm.app. Ціни Appointer — уточнити актуальні з 01.05.2025. Clinica Web — актуальне ціноутворення на clinica-web.ua.*
