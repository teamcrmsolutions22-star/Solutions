# CRM/ERP/SFA для оптової торгівлі та дистрибуції в Україні (2025–2026)

> **Розмежування класів систем — читай перед усім:**
> - **CRM** (KeyCRM, KeepinCRM, SalesDrive, Kommo, NetHunt, Uspacy) — воронки, комунікації, замовлення, базовий склад. Не замінюють ERP.
> - **ERP / облік зі складом** (Perfectum, Dilovod, Odoo, SAP B1, Dynamics 365, Skynum, IT-Enterprise) — склад, партії/терміни придатності, дебіторка, кредитні ліміти, прайс-листи, ПРРО, офіційні документи.
> - **SFA** (ABM Digital Distribution, Агент Плюс, MobileSOP) — польові продажі / маршрути / мерчандайзинг. Надбудова над ERP, не замінює ні CRM, ні бухгалтерію.
> - **SCM/AI-запаси** (ABM Inventory) — прогноз попиту, авто-поповнення. Теж надбудова над ERP.
> - **B2B-портал дилерів** (ClouB, Odoo native, Skynum) — самообслуговування дилерів, замовлення онлайн.

## 1. Для кого ця ніша

**Сегменти:**
- **A. Оптова торгівля / дистрибуція FMCG** — дилери, прайси по сегментах, повторні замовлення, дебіторка з відстрочкою
- **B. Дистрибутор з торговими представниками (SFA)** — польові продажі, маршрути, мерчандайзинг, план/факт по торгпредах
- **C. Імпортер / B2B-постачальник** — ЗЕД, мультивалюта, контракти, склад, інвойсинг
- **D. Складська логістика / 3PL фулфілмент** — зберігання, обробка замовлень, WMS, AI-запаси
- **E. B2B e-commerce / портал самообслуговування дилерів** — онлайн-замовлення, особистий кабінет дилера

**Ключові болі ніші:**
- Прайс-листи по дилерах з різними цінами для різних сегментів (оптова / VIP / сезонна / індивідуальна)
- Дебіторка з відстрочкою + кредитні ліміти + блокування відвантаження при перевищенні
- Склад / залишки / резервування + партіонний облік (терміни придатності для FMCG/фарма)
- Торгпреди «в полі»: маршрути, замовлення з планшета, мерчандайзинг, GPS-контроль
- Реактивація сплячих дилерів та регулярні авто-замовлення
- Заборона 1С / BAS → міграція на безпечний UA або міжнародний стек

## 10. Підводні камені

### 10.1 ❌ Заборона BAS та 1С (обов'язково читай)

- **08-09.01.2026** — ДССЗЗІ опублікувала перелік з 27 забороненого ПЗ
- **10.02.2026** — додано ще 13 найменувань
- Офіційний перелік: cip.gov.ua
- **Заборонені BAS-продукти 🚫🇷🇺:** BAS ERP, BAS Управління торгівлею, BAS Управління холдингом, BAS Документообіг КОРП, BAS Бухгалтерія КОРП, BAS ЗУП
- ⚠️ **BAS ≠ безпечна UA-альтернатива 1С**. Платформа BAS — локалізована версія 1С:Підприємство 8 (рос. код). Назва «BAS» **навмисно не містить слова «1С»** — прикидається незалежним продуктом. Клієнти, що перейшли з 1С → BAS у 2022–2023 «тимчасово», тепер мають мігрувати вдруге.
- **Законопроєкт №13505** (рекомендовано Комітетом ВР до прийняття): штраф **2% від річного обороту** за використання забороненого ПЗ
- **Дедлайн переходу:** 1 січня 2030 року
- Низка інтеграторів відкрито заявили, що припинили продаж BAS після 24.02.2022

### 10.2 ❌ Bitrix24

- Санкції РНБО/НАЗК (Указ №227/2023) до **2033 року**
- Припинив роботу в Україні з **01.06.2023**
- Юрисдикція РФ: ризик витоку комерційних даних, вимоги спецслужб РФ до доступу
- «Сіре» використання (VPN, піратські версії): юридична відповідальність при введенні штрафів
- **B2B-кабінети на Bitrix24** (кейс «Школярик» тощо) — технічно існують, але несуть усі ризики вище
- Альтернативи: **Uspacy** або **NetHunt** — для CRM-шару

### 10.3 Пастки при виборі системи

| Помилка | Реальність |
|---------|-----------|
| «KeyCRM = CRM для опту з прайсами по дилерах» | KeyCRM **не має** матриці сегментних прайс-листів нативно. Хороший для e-com з мультивалютою, слабший для класичного B2B-опту |
| «KeepinCRM закриє склад + дебіторку» | Склад базовий; дебіторка (кредитні ліміти, акти звірки) — у ERP, не в KeepinCRM |
| «SalesDrive = ERP для дистрибутора» | SalesDrive — CRM для інтернет-магазину; дебіторка/ліміти в BAS/1С або Dilovod |
| «Kommo підходить для дистрибутора» | Kommo = messenger-first CRM; без складу, без прайсів по дилерах; потребує ERP-інтеграції |
| «Uspacy = повноцінна альтернатива Bitrix24 для дистрибуції» | Uspacy без складського модуля — критичне обмеження; підходить як CRM-шар |
| «OneBox = ERP» | OneBox — CRM-конструктор; регламентна бухгалтерія (ПДВ, декларації) потребує окремого Dilovod/BAS |
| «CRM замінить SFA для торгпредів» | CRM з мобільним застосунком ≠ SFA. Маршрути/GPS/мерчандайзинг — тільки в ABM DD, Агент Плюс, MobileSOP |
| «Odoo безкоштовний» | Community = без підтримки та підприємницьких функцій; Enterprise — від €11.90/user + послуги партнера |

## 11. Шляхи міграції з 1С / BAS

| Звідки | Куди | Кому підходить |

- **Облік паралельно:** Dilovod (€15–53/міс) — бухгалтерія по НП(С)БО окремо від операційного обліку

## 12. Чек-лист вибору CRM/ERP для опту/дистрибуції

```
МІНІМАЛЬНІ ВИМОГИ (без них — не варіант):
□ Мультисклад + автосписання при відвантаженні
□ Кілька типів цін / прайс-листів (для різних дилерів)
□ Дебіторка: відстрочка, нагадування, акт звірки
□ Документи: рахунок/видаткова/ТТН
□ Інтеграція з Новою Поштою (ТТН нативно)
□ Українська мова + юрисдикція поза РФ
□ НЕ BAS, НЕ 1С, НЕ Bitrix24

ДЛЯ FMCG/ДИСТРИБУЦІЇ З ТОРГПРЕДАМИ:
□ SFA: маршрути + GPS + мобільне замовлення + мерчандайзинг
□ ABM Digital Distribution або аналог (не CRM!)
□ Контроль план/факт по торгпредах
□ Мерчандайзинг-звіти + фото з полиці

ДЛЯ ІМПОРТЕРІВ/ЗЕД:
□ Мультивалюта нативно (авто-курси, інвойс в USD/EUR)
□ Облік по НП(С)БО/МСФЗ: Dilovod або «Облік UA» на Odoo
□ Контракти в USD/EUR + рахунки у форматі SWIFT

ДЛЯ B2B-ПОРТАЛУ ДИЛЕРІВ:
□ Дилер бачить СВІЙ прайс (не публічний)
□ Онлайн-залишки в реальному часі
□ Замовлення напряму у систему (без менеджера)
□ Документи / акти / рахунки в особистому кабінеті
□ Кредитний ліміт видно дилеру

ДЛЯ AI / ПРОГНОЗУ:
□ Потрібний ABM Inventory або Odoo 18/19 AI Demand Sense
□ Datup.ai — якщо вже є Odoo/Dynamics і потрібна точність 95%+
□ «Next-order prediction» — custom ML або Odoo AI, не «з коробки»
```
