# HubSpot — Навчання, впровадження, партнери

> Актуально: червень 2026 (Deep Research). Маркери: ✅ офіц. | ⚠️ вторинне

## Офіційні ресурси

- **HubSpot Academy:** сертифікації **безкоштовні**; free-акаунт = безліміт курсів + free tools. **5 мов** (EN, FR, DE, PT, ES; подекуди JP). Релевантні для впровадження: Sales Software, Marketing Hub Software, Revenue Operations, Inbound. ✅
- **Knowledge Base** (knowledge.hubspot.com), **Community** (community.hubspot.com), **Academy YouTube**. ✅
- **HUGs (HubSpot User Groups):** **100+ груп у 35+ країнах**; безкоштовні квартальні офлайн+віртуальні події; теми 2026 — Breeze AI, Customer Agent, атрибуція, звітність. ✅

## Впровадження та onboarding

- **Обов'язковий onboarding** для **Professional + Enterprise** (waive — через сертифікованого Solutions Partner, який робить впровадження). ✅
- **Орієнтовні fees (⚠️ блог, варіюється):** Sales/Service Pro **$1,500** / Ent **$3,500**; Marketing Pro **$3,000** / Ent **$7,000**. Для Starter — не потрібен. ⚠️
- **In-app guided setup** є поруч із платним onboarding. ✅
- ⚠️ Часу впровадження офіційно не зафіксовано (партнерське зазвичай 4–12+ тижнів).

## Сертифікації за роллю (⚠️)

- Sales Rep → **Sales Software Certification**
- Marketing Manager → **Marketing Hub Software + Inbound**
- Admin → Sales/RevOps + (партнери) **CRM Implementation Accreditation**

## Партнерська програма (Solutions Partners) — нова політика з 15.01.2026

- **Рівні:** untiered (Provider) → **Gold → Platinum → Diamond → Elite** (Elite — invite-only). ✅
- **Пороги** (sourced points / MRR / total threshold) ⚠️ (з політики через екстракти):
  - **Gold:** 110 sourced (~$2,200 MRR), threshold 325
  - **Platinum:** 325 sourced (~$6,500 MRR), threshold 925
  - **Diamond:** 950 sourced (~$19,000 MRR), threshold 3,100, ≥75% GRR (trailing 12 міс)
  - **Elite:** 2,100 sourced (~$42,000 MRR), threshold 9,000, ≥80% GRR, ~100 user-сертифікацій, invite-only
- **Сертифікація партнера:** оновлювати ≥1 співробітником щонайменше кожні **25 міс**; managed-points мінімум скасовано. ✅
- **Переваги:**
  - **Partner Resource Portal** (на Mindmatrix PRM) — усім рівням, у т.ч. untiered; **deal-registration** tool. ✅
  - **MDF** — Platinum/Diamond/Elite; до **50%** matching для lead-gen подій (через портал, регіональний review). ✅
  - Co-marketing — за tier/акредитаціями/базою/нішею. ✅
  - **NFR** (not-for-resale internal software) — стандартна перевага; конкретні умови ⚠️.
