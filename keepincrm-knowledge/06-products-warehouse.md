# KeepinCRM — Товари та Склад

> Налаштування товарів: https://keepincrm.com/general-products-settings  
> Функціонал складу: https://keepincrm.com/funktsional-skladu  
> Товари в угодах: https://keepincrm.com/products-in-deals  
> Типи цін: https://keepincrm.com/price-types  
> Правила авто-ціноутворення: https://keepincrm.com/autogenerated-price

---

## Каталог товарів

### Структура товару

| Поле | Опис |
|------|------|
| Назва | Повна назва товару |
| Артикул (SKU) | Унікальний код |
| Ціна | Базова ціна (роздрібна) |
| Собівартість | Закупівельна ціна |
| Кількість на складі | Залишки |
| Категорія | Група товарів |
| Опис | Текстовий опис |
| Фото | Зображення товару |
| Одиниця виміру | шт, кг, м тощо |

---

## Типи цін

Документація: https://keepincrm.com/price-types

Можна налаштувати кілька типів цін для одного товару:
- Роздрібна
- Оптова
- VIP / Особлива
- Будь-яка кастомна назва

**Налаштування:** Налаштування → Типи цін  
При додаванні товару в угоду менеджер обирає потрібний тип ціни.

---

## Правила авто-ціноутворення

Документація: https://keepincrm.com/autogenerated-price

Автоматичне формування ціни залежно від:
- Кількості (наприклад: від 10 шт — оптова ціна)
- Категорії клієнта
- Умов угоди

## Склад

Документація: https://keepincrm.com/funktsional-skladu

### Можливості складського обліку

| Операція | Опис |
|----------|------|
| **Оприбуткування** | Додавання товарів на склад |
| **Списання** | Ручне або автоматичне зменшення залишків |
| **Повернення** | Повернення товарів на склад |
| **Бронювання** | Резервування товарів під угоду |
| **Скасування бронювання** | Звільнення зарезервованих товарів |
| **Переміщення** | Між складами (якщо кілька) |

### Автоматичне списання

При підтвердженні угоди (зміна на певний етап):
- Тригер → автоматично списати товари зі складу
- Налаштовується через Автоматизацію

### Кілька складів

KeepinCRM підтримує роботу з кількома складами:
- Основний склад
- Склад відправлень
- Склад повернень
- Будь-які кастомні

## Товари в угодах

Документація: https://keepincrm.com/products-in-deals

### Як додати товар в угоду
1. Відкрити угоду
2. Перейти на вкладку "Товари"
3. Вибрати з каталогу або ввести вручну
4. Вказати кількість та тип ціни
5. Знижка (в % або в сумі)
6. Система автоматично розраховує суму

### Поля товару в угоді
- Назва / артикул
- Кількість
- Ціна (тип ціни)
- Знижка
- Сума рядка
- Загальна сума угоди

## Синхронізація залишків через Webhook

Документація: https://keepincrm.com/sending-products-movements

Вихідний webhook при зміні залишків:
- При кожному русі товару KeepinCRM відправляє дані на зовнішній URL
- Дозволяє синхронізувати залишки з сайтом або іншою системою

## Налаштування товарів

Документація: https://keepincrm.com/general-products-settings

**Шлях:** Налаштування → Товари та послуги

- Увімкнення/вимкнення складського обліку
- Одиниці виміру
- Категорії
- Кастомні поля для товарів
- Правила авто-списання
