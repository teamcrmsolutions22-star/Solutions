# KeepinCRM — Автоматизація та Тригери

> Документація: https://keepincrm.com/automation  
> Тригери: https://keepincrm.com/main-triggers  
> Автоматичні завдання: https://keepincrm.com/create-tasks

---

## Що таке тригери?

**Тригери** — інструмент автоматизації в KeepinCRM, який виконує дії автоматично при настанні певних умов. Налаштовується один раз і далі працює без участі менеджера.

> ⚠️ **Автоматизація доступна лише на розширеному тарифі**

---

## Де налаштовуються

**Налаштування → Автоматизація → Тригери**

---

## Для яких розділів доступні тригери

- ✅ Ліди
- ✅ Клієнти
- ✅ Угоди (замовлення)
- ✅ Завдання
- ✅ Доставки
- ✅ Дзвінки
- ✅ Фінанси

---

## Структура тригера

Кожен тригер складається з:

1. **Назва** — ім'я тригера для розуміння
2. **Розділ** — до якого розділу відноситься (угоди, ліди тощо)
3. **Умова** — при якій події спрацьовує
4. **Дія** — що виконати

---

## Умови (тригери)

### Загальні умови
| Умова | Опис |
|-------|------|
| Змінено статус | При зміні будь-якого поля статусу |
| Змінено відповідального | При зміні менеджера |
| Значення поля дорівнює | Кастомна умова по полю |
| Змінено етап | При переміщенні угоди |

### Умова по етапу воронки
```
stage.name == 'Назва етапу'
```

### Умова з декількома воронками
```
stage.name == 'Узгодити документи' and funnel.title == 'Продажі'
```

### Умова по полю
```
field.назва_поля == 'значення'
```

## Дії тригерів

### Завдання
- Автоматично створити завдання
- Призначити відповідального
- Авторозподіл (між групою менеджерів)

### Переміщення
- Перемістити на інший етап воронки
- Перемістити в іншу воронку
- Перемістити контрагента в архів
- Завершити угоду (виграно/програно)

### Поля
- Скопіювати значення поля
- Заповнити поле значенням

### Комунікація
- Надіслати SMS клієнту
- Надіслати Email клієнту
- Надіслати повідомлення в месенджер

### Документи та фінанси
- Згенерувати документ
- Провести фінансову операцію
- Створити ТТН

### Угоди
- Створити нову угоду (для допродажів)

## Тригер: переміщення між воронками

Окрема документація: https://keepincrm.com/transfer-funnel

Дозволяє автоматично перемістити угоду з однієї воронки в іншу при виконанні умови.

**Приклад:** При переході на етап "Виграно" в воронці "Лідогенерація" → автоматично створити угоду у воронці "Виконання"

---

## Вихідний Webhook як дія тригера

Тригер може відправити дані на зовнішній URL (вихідний webhook):
- При будь-якій події в системі
- Дані про угоду/контрагента у форматі JSON
- Можна налаштувати кастомні поля

Документація: https://keepincrm.com/send-from-deals

---

## Автоматичне створення завдань

Документація: https://keepincrm.com/create-tasks

Налаштування автоматичного призначення завдань при:
- Створенні нової угоди
- Переміщенні на певний етап
- Зміні відповідального

---

## Приклади налаштованих тригерів

### 1. Нова угода → Створити завдання "Зателефонувати"
- **Умова:** Створено нову угоду
- **Дія:** Створити завдання "Зателефонувати клієнту" на відповідального

### 2. Відправлення → SMS трекінг
- **Умова:** Змінено поле "ТТН" (заповнено)
- **Дія:** Надіслати SMS з номером ТТН клієнту

### 3. Оплата → Перейти на наступний етап
- **Умова:** Проведено оплату в повному обсязі
- **Дія:** Перемістити угоду на етап "Відправлено"

### 4. Програна угода → Архів клієнта через 30 днів
- **Умова:** stage.name == 'Програно'
- **Дія:** Перемістити контрагента в архів (через затримку)

### 5. Роzetka/Prom нове замовлення → Угода + Завдання
- **Умова:** Нова угода з джерела "Rozetka"
- **Дія 1:** Авторозподіл між менеджерами
- **Дія 2:** Створити завдання "Підтвердити замовлення"
