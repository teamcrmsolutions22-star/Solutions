| Показник | Опис |

---

## Розділ Аналітика

### По угодах/замовленнях

**Вкладка «По менеджерах»:**
- Кількість угод кожного менеджера
- Сума угод
- Конверсія
- Середній час обробки
- Виграно / Програно

### По лідах та клієнтах
- Загальні параметри
- Конверсія лід → клієнт
- Інформація про джерела
- Розподіл по статусах
- Динаміка за період

### По джерелах
- Ефективність кожного каналу залучення
- ROI по джерелах
- Кількість та якість лідів

### По воронці
- Кількість угод на кожному етапі
- Час перебування на етапах
- «Вузькі місця» у воронці

## Фінансові звіти

| Звіт | Опис |
|------|------|
| Квартальний | Доходи/витрати за квартал |
| Грошовий потік | Cashflow по місяцях |
| Річний баланс | Підсумок за рік |
| PNL | Прибутки та збитки по статтях витрат |

---

## Аналітика комунікацій

- Канали комунікацій (месенджери, дзвінки)
- Оператори (ефективність менеджерів у чатах)
- Час відповіді
- Кількість оброблених чатів

---

## Статистика складу

- Рух товарів (оприбуткування / списання)
- Залишки на дату
- Товари, яких немає в наявності
- Найбільш продавані товари

---

## Фільтри аналітики

По всіх звітах доступна фільтрація:
- По даті (день / тиждень / місяць / квартал / рік / кастомний)
- По відповідальному / менеджеру
- По джерелу
- По воронці
- По категорії товарів
