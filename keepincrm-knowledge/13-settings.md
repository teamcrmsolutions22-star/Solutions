# KeepinCRM — Налаштування системи

> Початок роботи: https://keepincrm.com/lets-start  
> Загальні налаштування: https://keepincrm.com/category/settings

---

## Структура розділу «Налаштування»

### Профіль компанії
- Назва компанії
- ЄДРПОУ / ІПН
- Адреса
- Контактні дані
- Логотип
- **API** — ключ для інтеграцій (тільки для власника акаунту)

### Типи даних

**Воронки продажів:**
- Шлях: Налаштування → Типи даних → Угоди
- Створення воронок та етапів
- Кольори, назви, терміни перебування

**Статуси контрагентів:**
- Постійний, Новий, VIP, тощо
- Кастомні статуси

**Джерела:**
- Сайт, Instagram, Facebook, Prom, Rozetka, Телефон, тощо
- Кастомні джерела

**Категорії фінансів:**
- Статті доходів та витрат
- Кастомні категорії

### Кастомні поля

Для кожного розділу окремо:

| Розділ | Шлях |
|--------|------|
| Угоди | Налаштування → Поля → Угоди |
| Контрагенти | Налаштування → Поля → Контрагенти |
| Товари | Налаштування → Поля → Товари |

**Типи полів:**
- Текст (однорядковий)
- Великий текст (багаторядковий)
- Число
- Дата
- Список варіантів (dropdown)
- Прапорець (checkbox)
- Посилання (URL)
- Файл

### Права доступу та ролі
- Обмеження розділів меню для менеджерів
- Обмеження видимості угод (тільки свої / всі)
- Обмеження видимості клієнтів
- Обмеження доступу до налаштувань
- Ролі: Адміністратор, Менеджер, Кастомна

### Управління
- Джерела лідів та клієнтів
- Статуси контрагентів
- Відповідальні менеджери
- Команди / відділи

### Склад
- Увімкнення/вимкнення складського обліку
- Одиниці виміру (шт, кг, м тощо)
- Категорії товарів
- Налаштування автоматичного списання

### Прайс-лист
- Типи цін (Роздрібна, Оптова, VIP)
- Правила авто-ціноутворення

### Інтеграції
- Нова Пошта, Укрпошта, Justin, Meest
- Prom.ua, Rozetka, Allo
- monobank, MonoPay
- Checkbox (ПРРО)
- Facebook/Instagram
- E-chat (Viber/Telegram/WhatsApp)
- Telegram бот, Viber бот
- Телефонія (Ringostat тощо)
- Webhook (вхідні та вихідні)
- API

### Шаблони документів
- Рахунок на оплату
- Акт виконаних робіт
- Договір
- Накладна
- Кастомні шаблони (Word/HTML формат)

**Змінні в шаблонах:**
```
{{client.name}} — ім'я клієнта
{{client.phone}} — телефон клієнта
{{client.email}} — email клієнта
{{deal.title}} — назва угоди
{{deal.total}} — сума угоди
{{deal.date}} — дата угоди
{{company.name}} — назва компанії
{{company.address}} — адреса компанії
{{company.edrpou}} — ЄДРПОУ
{{products}} — таблиця товарів
```

### Автоматизація
- Тригери (умови та дії)
- Автоматичні завдання

### Очищення даних
Функція для видалення тестових даних:
- Клієнти та ліди
- Угоди
- Завдання
- Склад
- Прайс-лист

> ⚠️ Незворотна операція!

### Білінг
- Поточний тариф
- Оплата
- Кількість користувачів
- Історія платежів

## Профіль користувача (індивідуальні налаштування)

Клік на ім'я/фото → Профіль

### Налаштування сповіщень

| Тип | Налаштування |
|-----|-------------|
| Email-сповіщення | На email акаунту |
| Звукові | При нових подіях |
| Push браузер | Навіть при згорнутій вкладці |

**Тригери push-сповіщень:**
- Нова угода
- Новий клієнт / лід
- Нове завдання
- Оновлення завдання
- Нове повідомлення в чаті

**Як увімкнути push:**
1. Профіль → Налаштування → Push-сповіщення
2. Натиснути «Отримати сповіщення»
3. Браузер запитає дозвіл → «Allow»
4. Якщо браузер заблокував → Налаштування браузера → Notifications → дозволити для keepincrm.com

### Гарячі клавіші
`Alt+Shift+[клавіша]` — швидке створення:
- Угоди
- Ліда
- Завдання
- та інших об'єктів

## З чого почати налаштування (порядок)

1. **Профіль компанії** — заповнити реквізити
2. **Воронка продажів** — налаштувати етапи під бізнес-процес
3. **Кастомні поля** — додати необхідні поля для угод та клієнтів
4. **Джерела** — перелік каналів залучення
5. **Статуси клієнтів** — якщо потрібні (Новий, VIP тощо)
6. **Інтеграції** — підключити Нову Пошту, маркетплейси, оплату
7. **Шаблони документів** — налаштувати рахунки, акти
8. **Автоматизація** — налаштувати тригери
9. **Права доступу** — обмежити доступ менеджерів
10. **API ключ** — для власних інтеграцій
