## Швидкий старт

1. Реєстрація на keepincrm.com (безкоштовний тариф для 1 користувача)
2. Налаштування профілю компанії: Налаштування → Профіль компанії
3. Налаштування воронки продажів: Налаштування → Типи даних → Угоди
4. Підключення інтеграцій: Налаштування → Інтеграції
5. Отримання API ключа: Налаштування → Профіль компанії → API
