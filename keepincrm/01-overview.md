# KeepinCRM — Загальний огляд

## Що таке KeepinCRM?

KeepinCRM — українська CRM-система для малого та середнього бізнесу, розроблена українськими фахівцями. Орієнтована переважно на товарний бізнес та e-commerce. Особливий акцент зроблено на інтеграціях з українськими сервісами: маркетплейси, доставка, оплата.

**Офіційний сайт:** https://keepincrm.com/  
**Документація:** https://keepincrm.com/instruktsiya-dlya-koristuvacha  
**API:** https://keepincrm.com/category/api  
**Webhook:** https://keepincrm.com/category/webhook  

---

## Що є в системі (єдиний простір)

- Ліди та клієнти (контрагенти)
- Угоди (замовлення) з воронками продажів
- Завдання
- Склад та товари
- Фінанси (доходи, витрати, банківські транзакції)
- Документи (рахунки, акти, договори)
- ТТН (Нова Пошта, Укрпошта, Justin, Meest)
- Чати (Instagram, Facebook, Viber, Telegram, WhatsApp)
- SMS розсилки
- Email
- Телефонія
- Автоматизація (тригери)
- Аналітика та звіти
- Інтеграції з зовнішніми сервісами

## Тарифні плани

| Параметр | Безкоштовний | Розширений |
|----------|-------------|------------|
| Кількість користувачів | 1 | Необмежено |
| Ліди та клієнти | ✓ | ✓ |
| Угоди | ✓ | ✓ |
| Завдання | ✓ | ✓ |
| Автоматизація (тригери) | ✗ | ✓ |
| Розширені інтеграції | Обмежено | ✓ |
| Чати (месенджери) | ✗ | ✓ |
| Вартість | 0 грн | ~349 грн/міс за 1 додаткового користувача |

> **Важливо:** На відміну від інших CRM, KeepinCRM пропонує фіксовану вартість без обмежень за кількістю користувачів на розширеному тарифі.

## Ключові переваги

1. **Українська розробка** — орієнтована на локальний ринок
2. **Нативні інтеграції** — Нова Пошта, Укрпошта, Justin, Meest, Prom, Rozetka, monobank, checkbox
3. **Просте налаштування** — інтуїтивний інтерфейс
4. **Гнучка воронка** — необмежена кількість воронок та етапів
5. **Повний цикл** — від ліда до відправки та оплати в одній системі
6. **Автоматизація** — тригери для будь-яких подій
7. **Безкоштовний старт** — план для 1 користувача без обмежень по часу

---

## Недоліки / Обмеження

- Каталог товарів — обмежені можливості кастомізації
- Автоматизація лише на розширеному тарифі
- Масові розсилки через месенджери заборонені (ризик блокування)
- Міжнародні маркетплейси не підтримуються

## Для кого підходить

- Інтернет-магазини (Prom, Rozetka, власний сайт)
- Оптові торговці
- Сфера послуг
- Малий та середній бізнес в Україні
- Бізнеси з товарними відправками Новою Поштою

---

## Де налаштовується

- **Налаштування** → **Профіль компанії** — основні дані компанії, API ключ
- **Налаштування** → **Типи даних** — воронки, статуси, поля
- **Налаштування** → **Інтеграції** — підключення зовнішніх сервісів
- **Налаштування** → **Автоматизація** → **Тригери** — автоматизація процесів
- **Налаштування** → **Шаблони документів** — шаблони для генерації
