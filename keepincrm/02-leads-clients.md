## Концепція контрагентів

У KeepinCRM контрагенти поділяються на два типи:

| Тип | Опис |
|-----|------|
| **Лід** | Потенційний клієнт, який ще не здійснив покупку |
| **Клієнт** | Контрагент, з яким вже є угоди або постійні відносини |

Переміщення між статусами відбувається автоматично або вручну.

---

## Структура картки контрагента

### Основні поля
- **Ім'я / Назва компанії** — повне ім'я або назва
- **Телефон** — один або декілька номерів
- **Email** — один або декілька
- **Джерело** — звідки прийшов контрагент
- **Відповідальний** — менеджер, закріплений за контрагентом
- **Теги** — для сегментації
- **Нотатки** — текстові нотатки
- **Статус** — лід/клієнт/архів

### Додаткові вкладки картки
- **Угоди** — всі угоди контрагента
- **Завдання** — завдання, пов'язані з контрагентом
- **Чати** — листування в месенджерах
- **Дзвінки** — історія дзвінків (при підключеній телефонії)
- **Документи** — документи контрагента
- **Фінанси** — фінансові операції
- **Нотатки** — хронологія нотаток

---

## Перегляди списку контрагентів

- **Таблиця** — стандартний список з колонками
- **Канбан** — картки по статусах/етапах
- **Картка** — візуальне відображення

---

## Фільтри та пошук

Фільтрація доступна по:
- Статусу (лід / клієнт / архів)
- Джерелу
- Відповідальному
- Тегам
- Датам (створення, оновлення)
- Кастомним полям

Фільтри можна зберігати для швидкого доступу.

---

## Кастомні поля

Можна додати будь-які додаткові поля для контрагентів:
- Текст
- Число
- Дата
- Список (dropdown)
- Прапорець (checkbox)
- Посилання

Налаштування: **Налаштування → Типи даних → Контрагенти**

---

## Операції з контрагентами

- **Створення** вручну або через форму/API/інтеграцію
- **Редагування** прямо зі списку або з картки
- **Злиття** дублікатів
- **Архівування** неактивних
- **Масові дії** — зміна відповідального, тегів, статусу для групи
- **Імпорт** з CSV/Excel
- **Експорт** в CSV/Excel

---

## Джерела контрагентів

Джерела налаштовуються в: **Налаштування → Типи даних → Джерела**

Можливі джерела:
- Ручне введення
- Форма на сайті (через Webhook або API)
- Instagram / Facebook
- Prom.ua
- Rozetka
- Telegram бот
- Телефонний дзвінок (телефонія)
- Inbound Webhook (будь-який зовнішній сервіс)

## API для контрагентів

### Створення контрагента (POST)
```
POST https://api.keepincrm.com/v1/counterparties
X-Auth-Token: {ваш_api_ключ}
Content-Type: application/json
```

Основні параметри запиту:
```json
{
  "name": "Ім'я клієнта",
  "phone": "+380501234567",
  "email": "client@example.com",
  "source_id": 1,
  "responsible_id": 1,
  "lead": true
}
```

Повна документація: https://keepincrm.com/category/api
