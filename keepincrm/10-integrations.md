## Доставка

### Нова Пошта (Nova Poshta)
- **Документація:** https://keepincrm.com/integratsiya-z-servisom-nova-poshta
- **Налаштування:** Налаштування → Інтеграції → Нова Пошта
- **Потрібно:** API ключ з особистого кабінету НП (Налаштування → Безпека)

**Можливості:**
- Створення ТТН прямо в угоді
- Підтягування міст та відділень з НП
- Автоматичне відстеження статусу посилки
- Кілька акаунтів НП одночасно
- Автоматичне заповнення даних клієнта при створенні ТТН
- Вибір параметрів: відділення/кур'єр, вид вантажу, платник, форма оплати
- Автоматичне відправлення SMS клієнту з номером ТТН

**Параметри налаштування:**
- Метод доставки (відділення/кур'єр/поштомат)
- Платник за доставку (відправник/одержувач)
- Опис вантажу (одяг, взуття тощо)
- Відділення відправки
- Вид вантажу (посилка/вантаж)
- Платник комісії за післяплату
- Місто відправки

## Маркетплейси

### Prom.ua
- **Документація:** https://keepincrm.com/integratsiya-prom
- **Налаштування:** Налаштування → Інтеграції → Prom

**Можливості:**
- Автоматичний імпорт замовлень з Prom в KeepinCRM
- Синхронізація статусів замовлень
- Завантаження існуючих замовлень (база клієнтів)
- Автоматичне створення ліда/угоди при новому замовленні

### Rozetka
- **Документація:** https://keepincrm.com/integratsiya-z-rozetka
- **Налаштування:** Налаштування → Інтеграції → Rozetka

**Можливості:**
- Автоматичний імпорт замовлень з Rozetka
- При зміні статусу угоди на "Відправлено" → автоматично відправляє ТТН в Rozetka
- Синхронізація статусів

## Оплата та Банки

### monobank
- **Документація:** https://keepincrm.com/monobank
- **Налаштування:** Налаштування → Інтеграції → monobank

## Фіскалізація

### Checkbox (ПРРО)
- **Документація:** https://keepincrm.com/checkbox-integration
- **Налаштування:** Налаштування → Інтеграції → Checkbox

**Підключення:**
1. Додати профіль Checkbox
2. Додати касира (PIN код + ліцензійний ключ)

**Можливості:**
- Фіскалізація чеків прямо з угод KeepinCRM
- Видача фіскального чеку клієнту

## Месенджери та Чати

### Instagram + Facebook
- **Документація:** https://keepincrm.com/instagram-and-facebook-connect

**Можливості:**
- Підключення сторінок Instagram та Facebook
- Чат з клієнтами в розділі Чати або з картки контрагента
- Всі коментарі до сторінки стають чатами
- Згадки в Instagram → автоматично створюється чат
- Прив'язка чату до угоди/контрагента

### E-chat (Viber, Telegram, WhatsApp особисті номери)
- **Документація:** https://keepincrm.com/e-chat

**Можливості:**
- Підключення особистих номерів Viber, Telegram, WhatsApp
- Листування з клієнтами без виходу з CRM
- Прийом вхідних та відправлення вихідних
- Масові розсилки ЗАБОРОНЕНІ (ризик блокування номера)

### Telegram Бот
- **Документація:** https://keepincrm.com/telegram-bot

Підключення Telegram бота для автоматичних відповідей та збору лідів.

## Налаштування нової інтеграції (загальний порядок)

1. **Налаштування → Інтеграції** → знайти потрібну інтеграцію
2. Натиснути "+" або "Підключити"
3. Ввести API ключ або авторизуватися через OAuth
4. Налаштувати параметри (воронка, відповідальний, джерело)
5. Зберегти та протестувати

| Prom.ua | Кабінет продавця → API |
| Rozetka | Кабінет продавця → Налаштування |
| Checkbox | Кабінет Checkbox → Ліцензійний ключ + PIN касира |
