# KeyCRM — Огляд, ціноутворення, позиціонування

> Джерело: Аналіз YouTube-каналу (240 відео, 2021–2026), 2026-06-04

## Бізнес-модель та ціноутворення
- **$19/місяць** — фіксована базова оплата
- Оплата **за обсяг використання**, не за кількість користувачів
- Необмежена кількість користувачів
- **30 днів безкоштовного тріалу**
- Один тариф — весь функціонал включено
- Ціни в UAH

## Архітектура продукту
- **Cloud-only SaaS** — локальне розгортання не передбачено
- **5 ключових модулів:** Замовлення ↔ Контакти ↔ Воронки ↔ Склад ↔ Фінанси
- Автоматична дедуплікація контактів за телефоном/email
- Товари: `productId` + `offerId` (варіант)
- Pipeline-картки конвертуються в замовлення зі збереженням усіх даних

## Позиціонування vs конкуренти
**Від яких систем мігрують клієнти:**
- AmoCRM, Bitrix24 (основний потік після 2022 — де-русифікація)
- RetailCRM
- Pipedrive, Zoho, HubSpot, Odoo (занадто дорогі)

**7 ключових переваг:**
1. Ціни в UAH, без оплати за користувача
2. Вбудовані інтеграції — не маркетплейс плагінів
3. Українська тех. підтримка
4. Сервери в Європі (Амстердам + Фінляндія)
5. Налаштування без програмістів за 15 хвилин
6. Підходить і для e-commerce, і для послуг
7. Відкритий API
