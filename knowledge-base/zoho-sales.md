# Zoho CRM — Для Продажників та Партнерів

## Ключові Selling Points

1. **Zoho One** — 35+ продуктів, один логін, один рахунок. Головний диференціатор vs. точкові рішення.
2. **Відкрита архітектура** — API-first; HubSpot, Marketo, Google, зовнішні DB; не locked-in
3. **Масштабованість** — plug-and-play для 1 людини і enterprise-grade кастомізація
4. **Партнерська цінність** — партнери будують людські відносини, які запобігають SaaS churn; Zoho не може це зробити для 30M+ користувачів
5. **CRM як хребет бізнесу** — CRM Plus: маркетинг + продажі + проекти + підтримка + feedback
6. **Advanced Analytics** — multi-source blending, %, drill-downs
7. **AI автоматизація** — Zia (голос), Zobot (70% chat deflection), AI scoring
8. **Вертикальні рішення** — партнери будують галузеві продукти на базі Zoho

## Zoho One Pricing
- ~$30-45/user/month; всі 35+ додатків включені
- Партнери: "democratized pricing" — раніше потрібно було кілька вендорів

## Екосистема
- **CRM Plus:** 8-app suite для customer lifecycle
- **Bigin:** entry-level CRM для мікробізнесу (тільки воронка)
- **Creator:** low-code; кастомні портали, мобільні польові додатки
- **Catalyst:** serverless; AutoML, OCR, Face Analytics, Text Analytics
- **Agent Studio:** multi-agent builder; Digital Employees через Zoho Directory
- **Zoho MCP:** open protocol для multi-vendor agentic систем

## 6 Ключів Успішного Впровадження CRM (Susan Clark)

1. **Спочатку Vision** — сплануй звіти до конфігурації; думай про майбутні додатки
2. **User-friendly дизайн** — однаковий порядок полів; перша колонка = клікабельна; остання = власник
3. **Custom Views (найважливіший навик для навчання):**
   - All Open Potentials; Past Due Potentials; Past Due Activities; Today's Activities (OR логіка)
   - Клонуй views по продавцям; тимчасові views для масових оновлень
4. **Безкоштовні функції:** шаблони, мобільний додаток, Report Scheduler, mail merge
5. **Додавай додатки поступово:** Campaigns першим, потім Projects, Support, Books
6. **Навчай команду:** carrot не stick; максимум 5 речей за сесію; CRM champion; cheat sheets
