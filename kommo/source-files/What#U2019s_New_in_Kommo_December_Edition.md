# What’s New in Kommo | December Edition

welcome to the December edition of What's New in Como This month we're introducing a redesigned left menu Cleaner faster and easier to use It now offers key improvements Clear sections Tools are grouped into logical hubs like communication pipelines AI and automation and insights Consistent navigation Hovering and clicking behave the same across all areas Collapsible design Expand for full detail or keep it slim to maximize workspace Smarter settings Profile workspace and billing are now centralized for easier account management We also improved how you communicate on WhatsApp Introducing test mode a safe way to test your WhatsApp broadcast messages before going live Simply scan the QR code send a quick message and a test source will automatically connect to your account Launch the broadcast to yourself and watch the full sending flow in action Another important update we have is that you can choose whether the sender's name appears in the chat So when a CRM user writes to a customer the customer now sees the sender's name from the user profile Now let's talk about AI updates We are introducing follow-up messaging The AI agent can now send follow-up messages even without an incoming prompt You can set a condition and the agent will either send a message or perform another CRM action such as updating fields changing stages or launching bots Next a quick update on Como Tik Tok integration You can now configure Tik Tok chat prompts directly in the Tik Tok integration widget making it easier to create and edit interactive prompts without leaving Como For more information visit the COMO support and blog page See you next month for more updates
