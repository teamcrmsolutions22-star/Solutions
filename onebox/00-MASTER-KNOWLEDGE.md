## Що таке OneBox (ключові факти)

- **Українська no-code «операційна система для бізнесу»** — не одна програма, а **набір із сотень компонентних застосунків (apps)**, з яких збирається власне рішення (CRM / ERP / облікова система).
- Кожен застосунок **кастомізується і програмується без коду**; партнер/користувач може створювати власні застосунки (>1000 у каталозі).
- Кросплатформність (Windows, macOS, iOS, Android, веб) + мобільний застосунок **Box mobile**.
- Ключовий застосунок автоматизації — **Workflows** (конструктор бізнес-процесів).
- Дві лінійки: **OneBox OS** (актуальна, увесь розвиток) та **OneBox MVP** (стара, лише критичні виправлення).
- Сайти: **1b.app** (актуальний), `crm-onebox.com`.

## Швидкий довідник для інтегратора

- **API-ключ:** пароль у форматі **md5** від контакту з правами адміністратора (Користувачі/Співробітники → доступ) або генерується у застосунку API. Рекомендовано окремий **віртуальний адмін-співробітник** для запитів — інакше API «вибиває» живого користувача із сесії й займає робочу сесію. Деталі → `06-api.md`.
- **Бізнес-процес = «заказ/процесс»**: етапи + дії (тригери) + процедури; підпроцеси й батьківські процеси. Деталі → `03-business-processes.md`.
- **OneBox Network** — обмін БП/документами/платежами/контактами між різними інсталяціями (холдинги, клієнт↔постачальник).
- **Інтерфейс стандартизований:** фільтри ліворуч, пошук на одному місці, шестерня ⚙️ → налаштування застосунку; кастомні вкладки і мобільна сітка.
- **Продуктивність:** паралельна обробка по ядрах (імпорт прайсів, пошук, БП); пошук по 16 млн контактів < 3 c (⚠️ з відео).

## Відомі прогалини / на перевірку

- Точні назви хмарних тарифів понад базовий ($0 / індивідуальний) і модель «pay-per-data» — анонсована у відео, актуальні умови звіряти на 1b.app.
- AI-можливості: в офіційних матеріалах, що оглянуто, не виділено окремо — `⚠️ потребує перевірки`.
- Кілька транскриптів були майже нечитабельні (презентація OS, мобільна карточка продукту, Viber/TurboSMS, імпорт процесів XLS з вибором колонок) — відповідні розділи мінімальні.
