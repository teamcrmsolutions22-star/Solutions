# OneBox — Онбординг та базове налаштування

> Джерело: YouTube-транскрипти OneBox (автоматичні субтитри), дистиляція 2026-06-09

---

## 1. Встановлення та активація хмарного OneBox

### Реєстрація

1. Перейдіть на сайт **crm-onebox.com**.
2. На головній сторінці заповніть форму — всі поля обов'язкові:
   - Ім'я
   - Email
   - Номер телефону
   - **Адреса вашого OneBox** (доменне ім'я — саме за цим URL ви будете заходити в систему)
   - Пароль
3. Натисніть кнопку **«Отримати безкоштовно»**.
4. Зачекайте завершення сторінки завантаження — після цього вас перенаправить на головну сторінку вашого OneBox.
5. На вашу пошту прийде лист із логіном, паролем та посиланням для входу — збережіть його.

### Перший вхід і початкове налаштування

- Після редиректу система попросить вказати:
  - **Базову валюту** системи
  - **Назву компанії**
- Натисніть **«Почати»** — система готова до роботи.

## 2. Основні елементи OneBox OS

OneBox OS — операційна система для бізнесу, що складається зі **сотень застосунків**. Кожен застосунок налаштовується під конкретні вимоги.

### Ключові модулі (застосунки)

| Модуль | Призначення |
|--------|-------------|
| **Контакти** | База клієнтів, розподіл по групах, статистика, комунікація (email, SMS, дзвінки) |
| **Workflows** | Бізнес-процеси (ланцюжки етапів), налаштування та створення нових |
| **Платежі** | Вхідні/вихідні платежі, зарплата, відрахування, рахунки, мультивалюта |
| **Документи** | Рахунки, КП, акти — за шаблонами, автоформування в процесах |
| **Продукти** | База товарів і послуг, категорії, ієрархія |
| **Склади** | Необмежена кількість складів, оприбуткування, продажі, резервування, переміщення, переоблік |
| **Події** | Журнал усіх комунікацій — листи, SMS, дзвінки |

- Повний список застосунків — на сайті crm-onebox.com у розділі **«Застосунки»** та в модулі **OneBox Market** усередині системи.

## 3. Налаштування інтерфейсу замовлення / процесу

Інтерфейс кожного бізнес-процесу і кожного **окремого етапу** можна налаштовувати незалежно.

### Відкрити конструктор інтерфейсу

- **Спосіб 1:** Глобальне меню → **Workflows** → знайти потрібний БП → **Налаштувати** → **Конструктор інтерфейсу** → вибрати етап.
- **Спосіб 2:** Глобальне меню → Налаштування системи → Бізнес-процеси → вибрати БП → вибрати етап → кнопка **«Інтерфейс»**.

### Принципи роботи з конструктором

- Інтерфейс складається з **блоків**; блоки розміщені всередині **розмітки** (позначена штриховою лінією).
- Кожен блок містить набір **полів**.
- Блоки можна **переміщати**: затисніть ліву кнопку миші на заголовку блоку і перетягніть у потрібне місце розмітки.
- Для додавання нового блоку натисніть **«+»** всередині секції розмітки та оберіть тип блоку (наприклад, Універсальний блок).
- Налаштування блоку: натисніть кнопку **«Налаштування»** у блоці → у діалозі позначте галочками потрібні поля (наприклад, «Спосіб оплати») → **Зберегти**.

### Збереження інтерфейсу

- **Зберегти для поточного етапу** — натисніть кнопку **«Зберегти»**.
- **Зберегти для всіх етапів** цього бізнес-процесу — натисніть стрілку поряд з кнопкою → **«Зберегти для всіх етапів»** → OK.

## 4. Створення додаткових полів для процесу

Якщо стандартних полів недостатньо, можна створити власні.

### Крок 1 — Створити поле

1. Глобальне меню → **Налаштування системи** → **Прочее** → **Додаткові поля** → **Додаткові поля для процесів**.
2. Натисніть **«Додати поле»**.
3. Вкажіть **назву** поля та **тип** (рядок, список, дата тощо).
4. Натисніть **«Вставити новий запис»** — поле створено.

### Крок 2 — Вивести поле в інтерфейсі процесу

1. Перейдіть у конструктор інтерфейсу потрібного БП і етапу (див. розділ 3).
2. Знайдіть **Універсальний блок**, відкрийте його **Налаштування**.
3. Позначте галочкою нове поле → **Зберегти**.
4. Збережіть інтерфейс для поточного або всіх етапів.
5. Відкрийте процес, оновіть сторінку — нове поле відображається.

## 5. Налаштування фільтрів замовлень

### Налаштування колонок таблиці замовлень

1. Відкрийте застосунок **Замовлення**.
2. У правому верхньому куті натисніть **шестірню** → **«Колонки таблиці замовлень»**.
3. Позначте/зніміть галочки для потрібних колонок (наприклад, **Маржа**).
4. Змініть порядок колонок — перетягніть у потрібне місце лівою кнопкою миші.
5. Вкажіть кількість замовлень на сторінці (**не більше 100**, за замовчуванням 50).
6. Натисніть **«Зберегти»**. Можна зберегти тільки для себе або для всіх користувачів.

### Налаштування панелі фільтрів

1. У правому верхньому куті → шестірня → **«Панель фільтрації»**.
   - Або: Глобальне меню → Налаштування системи → **Прочее** → **Фільтри** → **Інтерфейс фільтрів бізнес-процесів**.
2. Видаліть непотрібні фільтри (кнопка «×» або хрестик).
3. Додайте нові — натисніть **«+»**, оберіть потрібний фільтр зі списку.
4. Змініть порядок фільтрів — перетягніть за **три крапки** зліва.
5. Натисніть **«Зберегти»** → перейдіть у список замовлень і оновіть сторінку для перевірки.

> Показ/приховання панелі фільтрів — кнопка-іконка зліва у списку замовлень.

## 6. Підключення пошти (email)

### Налаштування інтеграції з email (IMAP/SMTP)

1. Глобальне меню → **Налаштування системи** → **OneBox Market** → знайдіть **«Інтеграція з email»** → **«Налаштувати»** / **«Підключити»**.
2. Заповніть поля:
   - IMAP-хост, IMAP-порт
   - SMTP-хост, SMTP-порт
   - Email-адреса, пароль
3. Встановіть галочку **«Активний»**.

### Налаштування для Gmail

- IMAP і SMTP параметри — стандартні для Gmail.
- Додаткові дії в налаштуваннях Google-акаунта:
  1. Увімкнути **IMAP** у налаштуваннях Gmail.
  2. Вимкнути **двоетапну автентифікацію** (якщо увімкнена). ⚠️ потребує перевірки — Google посилив вимоги безпеки.
  3. У розділі **Безпека** → дозволити доступ **небезпечним застосункам**. ⚠️ потребує перевірки — Google може не підтримувати цю опцію у нових акаунтах.
  4. Перейти за спеціальним посиланням для надання доступу → натиснути **«Продовжити»**.
- Встановіть галочки: **«Не перевіряти сертифікат при підключенні до SMTP»** та **«Не відправляти листи з цієї пошти, якщо не доступний SMTP»**.
- За потреби — галочка **«Парсити назви папок з пошти»**.

### Результат після підключення

- Листи з пошти імпортуються в **Мої події** (глобальне меню).
- Якщо лист від невідомого контакту — автоматично створюється нова картка контакта.
- Переписка з контактом доступна в картці контакта на вкладці **«Події»**.

## 7. Підключення телефонії

Приклад: інтеграція з **Binotel** (been@tel).

### Кроки налаштування

1. Зареєструйтесь на **binotel.ua** (⚠️ назва сервісу відновлена з контексту).
2. Зверніться в **технічну підтримку Binotel** — отримайте **API-ключ** і **секрет/пароль**.
3. У OneBox: Глобальне меню → Налаштування системи → **OneBox Market** → знайдіть інтеграцію **Binotel** → **«Підключити»**.
4. Заповніть поля:
   - **Key** (API-ключ)
   - **Secret** (пароль)
   - **Номери** — внутрішні номери телефонії (наприклад, `901,902,903` або `9*` для всіх, що починаються на 9)
5. У картках контактів **співробітників** пропишіть їх внутрішні номери і вкажіть тип **Binotel**.
6. Відправте листа в технічну підтримку Binotel із шаблоном (доступний у описі інтеграції), де вказані:
   - Доменне ім'я вашого OneBox (повне, включаючи `https://`)
   - IP-адреса вашого сервера OneBox
7. Встановіть на комп'ютер **softphone** (програма-дзвонилка) або використовуйте стаціонарний IP-телефон — в ньому підключіть внутрішній номер, що відповідає номеру в картці співробітника.

### Здійснення дзвінка (Click-to-Call)

1. Відкрийте картку контакта → натисніть на номер телефону.
2. На softphone надійде вхідний дзвінок — відповісти на нього.
3. Система виконає вихідний дзвінок до клієнта; з'явиться спливаюче вікно з інформацією.
4. Після завершення дзвінок фіксується в **Моїх подіях**: ким дзвонили, кому, тривалість, запис розмови.

## 8. Підключення SMS-сервісу

Приклад: інтеграція з **TurboSMS** (turbosms.ua).

### Налаштування в особистому кабінеті TurboSMS

1. Увійдіть в особистий кабінет TurboSMS.
2. Відкрийте **Налаштування** → **API** → встановіть галочку **SOAP** (або відповідний тип API).
3. Заповніть:
   - Логін авторизації на сервері
   - Пароль авторизації на сервері (двічі)
4. Перейдіть у **Відправники** → **Відправники SMS** → створіть або оберіть існуючий підпис (sender name).

### Налаштування в OneBox

1. Глобальне меню → Налаштування системи → **OneBox Market** → знайдіть **TurboSMS** → **«Підключити»**.
2. Заповніть поля:
   - **API Логін** — логін авторизації на сервері
   - **API Пароль** — пароль авторизації на сервері
   - **Підпис (Відправник)** — підпис зі списку в TurboSMS (кілька — через кому)
3. Встановіть галочку **«Активний»** та **«За замовчуванням»** (якщо це основний SMS-сервіс).
4. Натисніть **«Підключити»**.

### Відправка SMS

- Відкрийте картку контакта → натисніть **SMS** поряд з номером телефону.
- Введіть текст → **«Відправити»**.
- Відправлене повідомлення відображається в **Моїх подіях**.

## 9. Підключення сервісу відправки email (SendPulse)

Приклад: інтеграція з **SendPulse** (sendpulse.ua / sendpulse.com).

### Налаштування в особистому кабінеті SendPulse

1. Увійдіть в особистий кабінет SendPulse.
2. Відкрийте **Налаштування акаунта** → вкладка **API** → активуйте **REST API** → отримайте **ID** і **Secret**.
3. Перейдіть на вкладку **SMTP** → вкажіть email, з якого буде відбуватись відправка листів (email проходить модерацію).

### Налаштування в OneBox

1. Глобальне меню → Налаштування системи → **OneBox Market** → знайдіть **SendPulse** → **«Підключити»**.
2. Заповніть поля:
   - **API ID** — ID з особистого кабінету
   - **API Secret** — секрет з особистого кабінету
   - **З яких email відправляти** — email, що пройшов модерацію (кілька — через кому)
3. Натисніть **«Підключити»**.
4. Після підключення листи з вказаних адрес відправлятимуться через SendPulse і відображатимуться в **Моїх подіях**.

## 10. Створення контактів та управління ними

### Створення контакту

1. Глобальне меню → **Контакти** → **«Додати фізичну особу»** або **«Додати юридичну особу/компанію»**.
2. Заповніть дані:
   - Тип контакту, ім'я, прізвище тощо
   - **Обов'язково**: телефон **або** email (принаймні одне контактне поле)
3. Натисніть **«Зберегти»**.

### Пошук і фільтрація контактів

1. Глобальне меню → Контакти → **Всі контакти**.
2. Натисніть кнопку **«Фільтрувати»** або сіру кнопку зліва — відкриється панель фільтрів.
3. Заповніть одне або кілька полів (тип, група тощо) → **«Фільтрувати»**.
4. Фільтрувати можна одночасно за необмеженою кількістю полів.

## 11. Розподіл контактів по групах

### Створення групи

- **Спосіб 1:** Застосунок Контакти → шестірня (налаштування) → **«Групи контактів»** → **«Створити групу контактів»** → вказати назву (і батьківську групу для ієрархії) → **«Вставити новий запис»**.
- **Спосіб 2:** Глобальне меню → Налаштування системи → Прочее → **«Групи контактів»** → **«Додати групу»** → заповнити → **«Вставити новий запис»**.

### Додавання контактів до групи

1. Відкрийте список всіх контактів, переключіться на вид **«Списком»**.
2. Позначте потрібні контакти галочками.
3. Відкрийте панель **масових операцій** → **«Додати до групи»** → оберіть групу → **«Зберегти»**.

### Налаштування інтерфейсу картки для кожної групи

- Інтерфейс картки контакта може бути **різним для кожної групи**.
- Щоб налаштувати: Застосунок Контакти → Налаштування → Групи контактів → вибрати групу → вкладка **«Інтерфейс»** (або кнопка «Інтерфейс»).
- Принцип роботи — аналогічний конструктору інтерфейсу процесу (блоки, поля, галочки).
- Зміни застосовуються до всіх контактів, що знаходяться в цій групі.

## 12. Створення додаткових полів для контакту

### Крок 1 — Створити поле

- **Спосіб 1:** Застосунок Контакти → Налаштування → **«Додаткові поля контактів»** → **«Додати»** → **«Додати поле»**.
- **Спосіб 2:** Глобальне меню → Налаштування системи → Прочее → **Додаткові поля** → **«Додаткові поля контактам»** → **«Додати поле»**.
- Вкажіть назву і тип поля (рядок, **випадаючий список** з варіантами — кожен з нового рядка, дата тощо).
- Натисніть **«Вставити новий запис»**.

### Крок 2 — Вивести поле в інтерфейсі картки контакту

1. Перейдіть у налаштування інтерфейсу потрібної **групи контактів** (розділ 11).
2. Знайдіть або додайте **Універсальний блок** → відкрийте його **Налаштування**.
3. Знайдіть нове поле (можна через швидкий пошук), позначте галочкою → **Зберегти**.
4. Збережіть інтерфейс групи → оновіть картку контакту з цієї групи — нове поле відображається.

> Можна створити необмежену кількість додаткових полів.

## 13. Email-розсилка по групі контактів

### Передумови

- Підключена інтеграція з email (розділ 6).
- Бажано підключений сервіс розсилки (наприклад, SendPulse, розділ 9).
- Контакти у групі мають прописаний email і встановлену галочку **«Підписаний на розсилку»**.

### Спосіб 1 — Через модуль «Масова розсилка»

1. Глобальне меню → **Розсилки** → **«Масова розсилка email»**.
2. Відкрийте панель фільтрації → оберіть **групу контактів**.
3. У полі **«Зворотна адреса»** вкажіть email відправника (має бути інтегрований в OneBox).
4. Заповніть **тему** і **текст** листа, **дату відправки** (якщо не вказати — відправиться одразу).
5. Натисніть **«Відправити»**.

### Спосіб 2 — Через застосунок «Масова розсилка» з плануванням

1. Застосунок **«Масова розсилка»** → **«Запланувати нову розсилку»**.
2. Вкажіть назву і дату старту → **«Додати»**.
3. На вкладці **«Шаблон розсилки»**: вкажіть email відправника, тему, текст листа → **«Зберегти»**.
4. На вкладці **«Параметри відправки»**: активуйте розсилку, вкажіть кількість листів на годину, дні тижня тощо → **«Зберегти»**.
5. Перейдіть у **«Список контактів»** → натисніть **«+»** → оберіть спосіб додавання:
   - Поштучно — вввести email
   - Перейти у застосунок Контакти → відфільтрувати → позначити → **«Додати в масову розсилку»** → вибрати розсилку → **«Зберегти»**
6. Після заповнення всіх вкладок без помилок — розсилку заплановано.
7. У списку розсилок відображається: кількість листів всього / відправлено / прочитано.

## 14. SMS-розсилка по групі контактів

### Передумови

- Підключена інтеграція з SMS-сервісом (наприклад, TurboSMS, розділ 8).
- Контакти у групі мають прописаний номер телефону і встановлену галочку **«Підписаний на розсилку»**.

### Кроки

1. Глобальне меню → **Розсилки** → **«Масова розсилка SMS»**.
   - Якщо інтеграція з SMS-сервісом не налаштована — отримаєте відповідну помилку.
2. Відкрийте панель фільтрації → оберіть **групу контактів** — розсилка виконується по контактах цієї групи.
3. За потреби оберіть **шаблон SMS** (налаштовується окремо) або просто введіть текст у відповідне поле.
4. Натисніть **«Відправити»** — розсилку запущено.

## 15. Фонова автоматика

Фонова автоматика — це дії, що **спрацьовують за розкладом** (незалежно від бізнес-процесів).

### Налаштування

- **Спосіб 1:** Глобальне меню → Налаштування системи → **Автоматизація**.
- **Спосіб 2:** Список застосунків → застосунок **«Автоматизація»**.

### Три підрозділи за частотою спрацювання

| Підрозділ | Частота |
|-----------|---------|
| Автоматичні дії 1 хвилину | Раз на хвилину |
| Автоматичні дії 1 година | Раз на годину |
| Автоматичні дії 1 день | Раз на день |

### Додавання дії

1. Відкрийте потрібний підрозділ (наприклад, **«Автоматика 1 година»**) → натисніть **«Налаштувати»** або **«+»**.
2. Зі списку доступних дій виберіть потрібну — перетягніть у праву область **або** натисніть на її назву → **«Додати»**.
3. Натисніть на назву дії — розгорнуться її **налаштування**, заповніть їх.
4. Натисніть **«Зберегти»**.

**Приклад дії:** «Сформувати групу контактів — всі клієнти, у яких за весь час було більше N бізнес-процесів» → налаштувати БП, кількість процесів, цільову групу контактів.

## 16. Автоматика для контактів

Спрацьовує при **створенні** або **збереженні** картки контакту.

### Налаштування

- **Спосіб 1:** Глобальне меню → Налаштування системи → **Прочее** → розділ CRM → **«Автоматизації для контактів»**.
- **Спосіб 2:** Список застосунків → застосунок **«Автоматизація для контактів»**.

### Два тригери

- **Дії при створенні картки контакту** — спрацьовує один раз під час створення.
- **Дії при збереженні картки контакту** — спрацьовує щоразу при збереженні.

### Додавання дії

1. Оберіть потрібний тригер → натисніть **«Налаштувати»** або **«+»**.
2. Зі списку оберіть дію → натисніть на її назву → **«Додати»** / **«OK»**.
3. Натисніть на **«Налаштування»** дії → заповніть параметри.
4. **«Зберегти»**.

**Приклад:** Дія «Заповнити поле картки контакту на основі іншого поля» → копіювати значення з поля `email` у поле `Коментар адміністратора`. Щоразу при збереженні картки — поле оновлюється автоматично.

## 17. Автоматика для процесів

Автоматизація для процесів — це **дії на етапах** бізнес-процесу, що спрацьовують при переключенні між етапами.

### Відкрити налаштування дій для етапу

- **Спосіб 1:** Застосунок **Workflows** → вибрати БП → у блок-схемі вибрати етап → кнопка **«Дії»**.
- **Спосіб 2:** Глобальне меню → Налаштування системи → **Бізнес-процеси** → вибрати БП → вибрати етап → кнопка **«Дії»**.

### Додавання дії на етап

1. Натисніть **«+»** для додавання нової дії.
2. Знайдіть потрібну дію у списку (можна через пошук) → натисніть → **«Додати»**.
3. Натисніть на назву дії — відкриються налаштування → заповніть → **«Зберегти»**.
4. Для наступного етапу — поверніться у блок-схему або оберіть інший етап у лівому верхньому куті.

### Приклади дій

| Дія | Ефект |
|-----|-------|
| **Переключити на етап миттєво** | При вході на поточний етап — миттєво переходить на зазначений |
| **Перевірити заповненість полів** | Блокує перехід на наступний етап, якщо поле не заповнене |

> На одному етапі можна встановити кілька дій або не встановлювати жодної.

## 18. Форум технічної підтримки

### Задати питання на форумі

1. Перейдіть на сайт **crm-onebox.com** та **авторизуйтесь** під тим email, на який зареєстровано ваш OneBox.
2. Перейдіть у розділ **«Підтримка»** → натисніть **«Задати своє питання»**.
3. Заповніть форму:
   - Категорія
   - Назва питання
   - Текст питання
   - Теги (за потреби)
   - Завантажте файли (за потреби)
4. Встановіть галочку **«Підтверджую, що задаю питання публічно»**.
5. Натисніть **«Задати питання»** — питання опубліковано на форумі.

### Підписатися на тему форуму

1. Перейдіть на сайт crm-onebox.com → авторизуйтесь.
2. Щоб переконатися в правильній авторизації: перейдіть на вкладку **«Мої OneBox»** — має відображатись доменне ім'я вашої системи.
3. Перейдіть у розділ **«Підтримка»** → знайдіть потрібне питання (за допомогою фільтрації по категоріях або вручну).
4. Відкрийте питання → натисніть кнопку **«Підписатися»** / **«Підписатися на питання»**.
5. Після підписки — при кожному новому коментарі у цій темі ви отримаєте **email-сповіщення**.
