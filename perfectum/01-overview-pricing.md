# Perfectum CRM+ERP — Огляд та тарифи

> Джерела: Gemini + ChatGPT Deep Research, верифіковано 2026-05-31
> Версія: Perfectum ERP 9.8

## Огляд системи

**Тип:** Комплексна ERP/CRM для малого та середнього бізнесу України. З 2017 року.
**Юзери:** 170 000+/місяць (в одних матеріалах) / 200 000+ (FAQ сайту) — ⚠️ маркетингова невідповідність.
**Мови:** 16, включно з українською.
**Документація:** 250+ технічних матеріалів.
**Розгортання:** Хмара платна, Хмара безкоштовна (без ліміту строку), Коробка (Self-Hosted).

## Конфігурації (версії)

| Версія | Призначення |
|--------|-------------|
| **PROJECT** | Завдання, Kanban, Ганта, тайм-трекінг |
| **CRM** | Ліди, клієнти, фінанси, договори, аналітика продажів |
| **RETAIL** | Склад, замовлення, POS, маркетплейси, доставки |
| **ERP** | CRM + Project + Retail в одному просторі |
| **MAX** | Все вище + всі галузеві рішення |
