# Perfectum CRM+ERP — Дебаг, Імпорт/Міграція

## Інструменти дебагу
- Лог операцій: Налаштування → Системні → Журнал операцій
- Лог вебхуків: Журнал операцій веб-хуків
- Заблоковані IP: Список заблокованих IP
- Журнал відкладеного імпорту

## Типові помилки

**401/403 Unauthorized**
- Причина: невірний ключ, IP не в білому списку, немає прав на модуль
- Рішення: перевірити токен, очистити IP для тестів, увімкнути права модуля

**400 Bad Request при редагуванні завдань/лідів**
- Причина: невірний `staff_assignee`, `rel_id`, `rel_type`
- Рішення: передавати відповідно до документації API v2.5

**429 Too Many Requests**
- Причина: перевищено мікро або добовий ліміт → тимчасове блокування IP
- Рішення: backoff strategy + кешування; для зняття лімітів → коробкова версія

**Бізнес-процес не спрацьовує**
- Причина: БП на паузі / не налаштований Cron / надто складні умови
- Рішення: зняти паузу, перевірити Cron, спростити тригер для тестування

**Помилка імпорту >300 записів**
- Причина: хмарне обмеження 300 записів за 1 операцію
- Рішення: «Відкладений імпорт» для файлів >300 рядків

## Імпорт / Експорт / Міграція

### Імпорт через UI
- Формат: Excel (.xlsx)
- Сутності: Ліди, Клієнти, Товари (Склад), Позиції прайсів
- **Хмарний ліміт: 300 записів за 1 операцію**
- Великі файли → «Відкладений імпорт» (черга, автозавантаження частинами)
- Дублікати: товари оновлюються за SKU; ліди/клієнти з дублями Email — ігноруються

### Експорт
- Формати: CSV або Excel
- Клієнти, товари, API-ключі, вебхуки, заблоковані IP

### Міграція з інших CRM
Автоматичного імпортера немає ⚠️. Ручний порядок:
1. Довідники (Категорії, Склади, Країни)
2. Клієнти / Контакти
3. Товари
4. Відкриті замовлення / Ліди

## Відомі невизначеності ⚠️

| Питання | Як перевірити |
|---------|---------------|
| HMAC-підпис вебхуків | Аналізувати заголовки тестового вебхука (шукати `X-Signature`) |
| КЕП (електронний підпис) | Запит до техпідтримки про Вчасно.КЕП або Дія.Підпис |
| Formula fields (no-code) | Спробувати в налаштуваннях поля або запитати підтримку |
| Конектор Zapier/Make | Шукати "Perfectum" у zapier.com і make.com |
| Tranzzo | Запит до техпідтримки |
| Список кастомних полів через API | Перевірити GET /api/custom-fields |
