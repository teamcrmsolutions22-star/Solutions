# Pipedrive — Налаштування без коду (Admin)

## Базові концепції (сутності)

| Сутність | Опис |
|----------|------|
| **Deal** | Центральна транзакційна сутність: сума, стадія, дата закриття, прив'язка до Person/Organization |
| **Person** | Фізична особа: телефони, email, історія взаємодії |
| **Organization** | Компанія-контрагент; до неї прив'язано багато Person |
| **Lead** | "Холодний" контакт у Leads Inbox, не перевантажує воронку; успадковує кастомні поля Deal |
| **Activity** | Запланована дія менеджера (дзвінок, зустріч, завдання). Кожна відкрита угода має мати хоча б одну |
| **Pipeline** | Набір послідовних стадій від першого контакту до закриття |
| **Stage** | Крок у воронці; для кожної стадії задається % ймовірності закриття |
| **Product** | Номенклатура з ціною та артикулом; додається до угоди |

## Перші кроки після реєстрації

1. Базові налаштування: часовий пояс, валюта, робочі години
2. Створити Pipelines + Stages (стадії — дії, а не стани: "Надіслано пропозицію", не "В процесі")
3. Налаштувати кастомні поля
4. Запросити користувачів, розподілити по командах
5. Підключити корпоративну пошту та календарі

## Кастомні поля (Custom Fields)

- Створюються окремо для кожної сутності: Deals, Persons, Organizations, Products
- 16 типів: текст, число, Monetary, дата, користувач, Single-option, Multiple-option тощо
- Поля можна зробити **обов'язковими на певних стадіях** — угода не переходить далі без заповнення
- [Enterprise] — гнучкі права доступу до полів на рівні груп видимості

## Воронки та Права доступу

**Декілька воронок** — доцільні при різних каналах (прямі, дистриб'ютори, партнери) або при upsell/cross-sell процесі.

**Visibility groups [Professional+]:** які записи бачить користувач (тільки свої угоди тощо)

**Permission sets [Enterprise]:** що може робити користувач (експорт, видалення, створення полів)

## Workflow Automation [Advanced+]

Принцип: **Тригер → Умова → Дія** (без коду, виконується на серверах Pipedrive)

**Тригери:** нова угода, зміна стадії, завершення активності  
**Дії:** створити активність, надіслати шаблон листа, оновити поля картки

| Тариф | Активних автоматизацій | Умов в одному ланцюжку |
|-------|------------------------|------------------------|
| Essential | 0 | — |
| Advanced | 50 | до 3 |
| Professional | 150 | до 10 |
| Power / Enterprise | 250 | до 20 |

## Аналітика та Insights [Professional+]

- Звіти за кастомними полями
- Відстеження індивідуальних та командних Goals
- Revenue forecast (на основі % ймовірності + дати закриття)
- Аналіз активності менеджерів
- Deal rotting — візуальний індикатор угод без активності (всі тарифи)
