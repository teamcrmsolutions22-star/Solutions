# Pipedrive — Оновлення та Дебаг

## Останні зміни (2024–2025)

- **Грудень 2024 — грудень 2025:** Впровадження Token-based Daily Budget API (нова система лімітів)
- **Webhooks v2:** Повноцінний запуск, замінює legacy v1 (міграція обов'язкова)
- **Custom UI Extensions:** Нові формати — Custom Panels, Floating Windows, Custom Modals (iframe + SDK)
- **OAuth scopes:** Нові `webhooks:read`, `webhooks:full`, `leads:read`, `leads:full`
- **API v2:** Міграція в процесі, курсорна пагінація

## Deprecations

- **Webhooks v1:** Legacy, тільки через старі API-методи. Рекомендується негайна міграція на v2.
- Публічний Roadmap не опублікований ⚠️

## Інструменти розробника

- **Developer Hub:** Панель для створення Marketplace-додатків, OAuth Client ID/Secret, Sandbox
- **Webhook log:** Візуальний моніторинг спроб доставки — Tools and integrations → Webhooks
- Лог безкодових автоматизацій — не задокументований ⚠️

## Типові проблеми та рішення

### 401 / 403 Unauthorized / Forbidden
**Причини:**
- Недійсний `api_token`
- OAuth-токен без потрібного scope (наприклад, спроба POST /leads без `leads:full`)
- Блокування через visibility groups або permission sets

**Рішення:** Перевірити токен, оновити OAuth-сесію, перевірити всі запрошені scopes.

### 429 Too Many Requests
**Причини:**
- Перевищено Burst Limit (2-секундне вікно)
- Вичерпано денний бюджет токенів

**Рішення:** Черга запитів + Exponential Backoff + читати заголовок `Retry-After`.

### Кастомне поле не оновлюється (PUT/PATCH)
**Причини:**
- Передано Label (назву поля) замість 40-символьного хеш-ключа
- Для Option-полів передано текст замість числового id опції

**Рішення:** `GET /v1/dealFields` → знайти хеш за назвою → використати в запиті.

### Вебхуки не надходять / підписка зникає
**Причини:**
- Сервер не відповів за 10 сек або повернув не 2XX → Ban System → блокування на 30 хв
- 3 дні без успішної доставки → автоматичне видалення підписки

**Рішення:** Відразу відповідати HTTP 200, фонова обробка тіла вебхука в асинхронній черзі.

## Відомі протиріччя між джерелами

| Питання | Застаріле (❌) | Актуальне (✅) |
|---------|--------------|----------------|
| Кастомні поля в API | Передавати Label або ID | 40-символьний хеш-код |
| Rate limits | 100 запитів / 10 сек | Daily Budget токенів + Burst 2-секундне вікно |
| Вебхуки та rate limits | Підпадають під API ліміти | Вихідні вебхуки виключені з rate limits |
| AI Scoring | Доступний на всіх тарифах | Тільки Professional та вище |
