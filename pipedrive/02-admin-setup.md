# Pipedrive — Налаштування без коду (Admin)

> Плани актуальні станом на липень 2025: Lite / Growth / Premium / Ultimate.

## Базові концепції (сутності)

| Сутність | Опис |
|----------|------|
| **Deal** | Центральна транзакційна сутність: сума, стадія, дата закриття, прив'язка до Person/Organization |
| **Person** | Фізична особа: телефони, email, історія взаємодії |
| **Organization** | Компанія-контрагент; до неї прив'язано багато Person |
| **Lead** | "Холодний" контакт у Leads Inbox, не перевантажує воронку; успадковує кастомні поля Deal |
| **Activity** | Запланована дія менеджера (дзвінок, зустріч, завдання). Кожна відкрита угода має мати хоча б одну |
| **Pipeline** | Набір послідовних стадій від першого контакту до закриття |
| **Stage** | Крок у воронці; для кожної стадії задається % ймовірності закриття |
| **Product** | Номенклатура з ціною та артикулом; додається до угоди (Growth+) |

## Перші кроки після реєстрації

1. Базові налаштування: часовий пояс, валюта, робочі години
2. Створити Pipelines + Stages (стадії — дії, а не стани: "Надіслано пропозицію", не "В процесі")
3. Налаштувати кастомні поля
4. Запросити користувачів, розподілити по командах (Teams — Premium+)
5. Підключити корпоративну пошту та календарі

## Кастомні поля (Custom Fields)

- Створюються окремо для кожної сутності: Deals, Persons, Organizations, Products
- 16 типів: текст, число, Monetary, дата, користувач, Single-option, Multiple-option тощо
- Поля можна зробити **обов'язковими на певних стадіях** — угода не переходить далі без заповнення
- Ліміти: Lite 30 / Growth 100 / Premium 300 / Ultimate unlimited
- Premium+ — гнучкі права доступу до полів на рівні груп видимості

## Воронки та Права доступу

**Пайплайни — unlimited на всіх планах.**  
Декілька пайплайнів доцільні при різних каналах (прямі, дистриб'ютори, партнери) або для upsell/cross-sell.

| Права | Де | Деталі |
|-------|----|--------|
| **Visibility groups** | Premium+ | Які записи бачить користувач (тільки свої угоди тощо) |
| **Permission sets** | Premium+ | Що може робити: експорт, видалення, створення полів |
| **Teams** | Premium+ | Угруповання менеджерів; командні ліміти і звіти |

## Workflow Automation (Growth+)

Принцип: **Тригер → Умова → Дія** (без коду, виконується на серверах Pipedrive)

**Тригери:** нова угода, зміна стадії, завершення активності, дата-тригери  
**Дії:** створити активність, надіслати шаблон листа, оновити поля, пост у Slack/Teams, задача в Trello/Asana, webhook, тригер Campaigns

**Ліміти — на компанію (shared pool):**

| Тариф | Активних автоматизацій | Умов в ланцюжку |
|-------|------------------------|----------------|
| Lite | 0 | — |
| Growth | 50 | до 3 |
| Premium | 150 | до 10 |
| Ultimate | 250 | до 20 |

## Аналітика та Insights

- Звіти за кастомними полями
- Відстеження індивідуальних та командних Goals (Premium+)
- Revenue forecast (Premium+)
- Subscription revenue report (Growth+)
- **AI Report Generation** — звіти через натуральну мову (Premium+, березень 2025)
- Deal rotting — візуальний індикатор угод без активності (всі плани)
- Live dashboard sharing з не-Pipedrive юзерами (всі плани)

**Ліміти звітів:** Lite 15/user, Growth 50/user, Premium 250/user, Ultimate unlimited.
