# Pipedrive — Міграція та імпорт даних

## Імпорт через UI

- Доступний на всіх тарифах
- Завантаження файлу + автоматичне маппування колонок на системні та кастомні поля
- Підтримувані формати, ліміти рядків, обробка дублів — не задокументовані ⚠️

## Порядок міграції (для збереження зв'язків)

```
1. Organizations  →  створити базу компаній
2. Persons        →  прив'язати до ID організацій
3. Deals          →  прив'язати до Person та Organization
4. Activities / Notes  →  прив'язати до відповідних угод
```

**Порушення порядку** → розбиті зв'язки між сутностями.

## Експорт

- Доступний на всіх тарифах через налаштування системи
- Full backup компанії в один клік — не задокументований ⚠️

## Нотатки

- При конвертації Lead → Deal всі накопичені дані автоматично переносяться
- Deal Products (додавання товарів через API) — окремий sub-endpoint, не через PUT угоди ⚠️
