# Planfix — Огляд платформи, тарифи, порівняння

## Що таке Planfix

Гнучкий low-code/no-code гібрид CRM + таск-менеджер + управління проєктами.  
**Ключова перевага:** абсолютна архітектурна пластичність — інструментарій конструктора без жорсткої нав'язаної логіки.  
**Ключове обмеження:** високий поріг входження, вимагає детального проектування бізнес-процесів.

- Запуск: травень 2009; відкрите бета: 24 червня 2010
- Аудиторія: мікро/малий/середній бізнес (SMB), Enterprise, некомерційні організації
- Домени: planfix.com, planfix.ua (→ planfix.com/uk/), planfix.ru
- Академія: academy-ua.planfix.com

## 5 Killer Features для демонстрації

1. **Безкодові сценарії автоматизації** — «Роботи повинні працювати!»
2. **Унікальний блок аналітики задач** — фінанси, час, ресурси прямо у тілі задачі
3. **Гнучкі дашборди в Planner** — Kanban, списки, ключові метрики
4. **Омніканальний комунікаційний хаб** — email, телефонія, соцмережі, месенджери в одному вікні
5. **Багаторівнева модель доступу** — field-level security
