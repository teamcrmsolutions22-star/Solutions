# Planfix — AI-можливості

## Вбудовані AI-функції

- **Автоматична обробка:** аналіз та категоризація вхідних звернень, створення задач з неструктурованого тексту, автозаповнення кастомних полів
- **Доступність:** лише на платних тарифах (Plan A / Plan B / Plan X) ✅

## AI-агенти

| Агент | Призначення |
|-------|------------|

## Рівні AI-моделі (з січня 2026)

| Рівень | Ціна | Коли використовувати |

| **Покращена** | Як раніше | Складний аналіз, документи, PDF |

## AI-агенти — деталі

### Датамайнер
- Аналізує коментарі контакту → заповнює поля задачі
- З січня 2026: заповнює «Запис довідника» (довідник ≤ 1000 записів)
- З січня 2026: аналізує PDF і зображення
- ⚠️ Аналізує коментарі **контактів**, не задач. Для файлів у задачі → «Послати запит до Planfix AI»

### Інтерв'юер
- Веде діалог → заповнює поля по мірі відповідей
- З січня 2026: не ставить питань по вже заповнених полях
- Налаштовується: «заповнювати якщо порожнє» або «завжди перезаписувати»

### Транскрибація (Speech Recognition)
- Кнопка «…» під аудіофайлом
- Швидкість: 3 хв → 10 сек; 10 хв → 30 сек; 1.5 год → 1 хв
- Мультиспікер

## Knowledge Base Search (База знань)

- Підтримувані формати: `txt`, `pdf`, `docx`
- Запущено: лютий 2026
- **Де:** Управління акаунтом → Налаштування Planfix AI → Пошук у базі знань
- **Режим: РУЧНИЙ** — кнопка «База знань» у коментарі → AI підказка → менеджер редагує ⚠️
- Для автоматичного бота: «Послати запит до Planfix AI» у сценарії + текст БЗ у промпті
- Працює як пошук за ключовими словами, **не семантичний**

## Аналіз дзвінків (Speech Recognition)

Поля після транскрибації: тон розмови, результат дзвінка, інциденти, рекомендовані наступні кроки

## Актуальний стан інтеграцій

| Метод | Статус |
|-------|--------|
| «Надіслати запит до **Planfix AI**» | ✅ Актуальний, рекомендований |
| «Надіслати запит до ChatGPT» | ❌ Статус **«As Is»**, більше не підтримується |

> **Важливо:** Стара дія ChatGPT офіційно переведена в статус «As Is» та не розвивається

## OpenAI (ChatGPT) Інтеграція

- Підключення: «Керування акаунтом» → «Інтеграції» → власний API-ключ OpenAI
- Модель за замовчуванням: **gpt-3.5-turbo**
- Результати розкладаються по кастомних полях або виводяться у коментарях

## AI через API

- Прямих ендпоінтів для виклику Planfix AI через зовнішній API **немає** ⚠️
- Обхідний шлях: зовнішній сервіс оновлює текстові поля задачі через REST API → запускається внутрішній сценарій → Planfix AI обробляє → повертає результат
