# Planfix — Віджети та програмні доробки

## Кастомні кнопки та дії

Адміністратор додає кнопки на панель картки задачі або контакту.

**Дії при натисканні:**
- Запуск автоматичного сценарію
- Відкриття зовнішнього iFrame-вікна
- Перехід за динамічним URL

**Контекст передачі даних:**
- GET-параметри: ID сутності, значення кастомних полів, токен авторизації

## Вбудовані сторінки та iFrame

- Сторонні веб-інтерфейси вбудовуються як кастомні вкладки або панелі планувальника
- **JavaScript SDK для iFrame → Planfix** не задокументований ⚠️
- Зв'язок реалізується через прямі REST API запити вбудованого додатка

## Обчислювані поля та формули

- **Математичні формули** для чисел та дат (тип поля «Розрахункове поле»)
- **Не підтримується:** виклик зовнішніх API з формули → для цього використовувати вебхуки в сценаріях

## Шаблони задач

- Автозаповнення полів на основі зв'язків з проектом або контрагентом
- Підтримка змінних при заповненні

## Маркетплейс

Каталог готових конфігурацій (рішень):
- Розгортають попередньо налаштовані поля, процеси, сценарії
- Готові рішення для різних бізнес-задач
- **Free тариф:** лише прості пакети (наприклад, «Робота з задачами»)
- **Paid тарифи:** будь-які складні конфігурації
