# Planfix — Звіти та Аналітики (детально)

> Джерело: YouTube-канал Planfix (транскрипти ~50 відео)

---

## Звіти (Отчеты)

### Типи об'єктів для звітів
1. **Дії і коментарі** — з задач і контактів разом
2. **Дії коментарів** — тільки з задач
3. **Дії коментарів в картці контакту** — тільки з контактів
4. **Задачі**
5. **Аналітики** — одна або кілька
6. **Аналітики в картці контакту** — з контактів
7. **Контакти і компанії**
8. **Довідники** — тільки один довідник
9. **Проекти**
10. **Календар відпусток**

### Стовпці (Столбцы)
- Налаштовуються на основі обраного об'єкту
- Можна відображати поля пов'язаних об'єктів (задача, проект, контрагент)
- Перетягування для зміни порядку
- Копіювання між стовпцями, вставка між існуючими
- Приховані стовпці — не видно, але доступні для групування/сортування
- Стовпці з формулами

### Агрегації (підсумки)
| Агрегація | Опис |
|-----------|------|
| Сума значень | Сума числових значень |
| Кількість елементів | Загальна кількість рядків |
| Кількість унікальних елементів | Без дублікатів |
| Середнє значення | Середнє |
| Максимальне | Найбільше |
| Мінімальне | Найменше |
| Перше значення | Перший рядок |
| Останнє значення | Останній рядок |
| Сукупність значень | Агрегація переліку |

### Групування
- Кілька рівнів вкладеності (1-й, 2-й рівень і т.д.)
- За системними полями або кастомними стовпцями звіту
- Дати: по днях, по тижнях, по місяцях, по роках
- При групуванні → агрегації відображаються в заголовках груп + рядок «Підсумок»

### Сортування
- Тільки за стовпцями звіту
- Напрям: зростання / спадання

### Фільтри (Параметри відбору)
- Множинні умови з логікою `І` / `АБО` і дужками
- Операторами: містить, не дорівнює, більше, менше, містить значення та ін.
- Зміна фільтру в реальному часі без збереження (кнопка «Показати параметри»)

### Формати виводу
- **Excel** (xlsx)
- **Google Таблиці**
- **CSV** з налаштованим роздільником
- **PDF** / друк
- Варіанти: видимі рядки на екрані або всі рядки

### Графіки та діаграми
| Тип | Опис |
|-----|------|
| Кругова діаграма | Частки |
| Воронка | Конверсія |
| Стовпчикова | Порівняння |
| Лінійна | Динаміка |
| Гістограма | Розподіл |

- Кілька графіків в одному звіті
- Позиція: зверху, знизу, зліва, справа від даних
- Висота налаштовується в пікселях

### Формули в звітах
- Стандартні операції: `+`, `-`, `*`, `/`
- Умовні функції: `якщо`, `або`, `і`
- Посилання на стовпці за літерою (D, E, F...)
- **Змінні НЕ працюють** в звітах — лише посилання на стовпці ⚠️
- Вкладені `IF`-оператори підтримуються
- Область застосування формули: для кожного рядка / для першого угрупування / тільки для рядків підсумків

## Історія статусів у звітах

- `Історія статусів — статус` — всі статуси, через які пройшла задача
- `Час у статусі загальний` — загальний час у статусі
- `Час у статусі робочий` — лише робочий час
- `Дата переводу` — дата/час зміни статусу
- `Наступний статус` — в який статус перейшла

> ⚠️ Час менше 1 хвилини — **не записується** в звітах

## Управління доступом до звітів

- За замовч.: тільки автор і адмін
- Можна надати доступ: всім співробітникам, конкретним співробітникам, конкретним контактам
- Контакти з базовим доступом — не можуть запускати звіти
- Кожен бачить дані відповідно до своїх прав на об'єкти
- Адміністратор завжди бачить всі дані

### Запуск звітів
- З картки задачі — з автоматичним параметром задачі
- З картки контакту — з автоматичним параметром контакту
- З картки проекту — попередньо налаштований звіт

## Аналітики (Аналитики) — унікальний інструмент

### Ключова відмінність від полів задачі
| Параметр | Кастомні поля | Аналітики |
|---------|--------------|-----------|
| Кількість записів | Одне значення | Кілька рядків (накопичення) |
| Зміна в часі | Без історії | Зберігає всі рядки |
| Складність | Просте | Кілька пов'язаних полів |
| Звітність | Обмежена | Повна структурована |

### Де доступні аналітики
- ✅ Задачі
- ✅ Контакти
- ❌ Проекти
- ❌ Співробітники
- ❌ Довідники

### Системна аналітика «Плановий час роботи»
- Завжди присутня, видалити неможливо
- Автозаповнення при: призначенні конкретного виконавця + конкретного дедлайну в межах одного дня
- Використовується для: годинного розкладу, тайм-трекінгу

### Додавання аналітик
- Через скрепку в коментарі
- При створенні задачі: вкладка «Прикріпити»
- В картці задачі: вкладка «Аналітики»
- Контакти: форма редагування → вкладка «Аналітики»

### Відображення рядків аналітик
- За замовч.: горизонтально
- Альтернатива: «переворот» (вертикальні стовпці)
- Налаштування застосовується до конкретного екземпляра

### Доступ до аналітик
- Обмеження за статусом: додавання, редагування, видалення
- За посадою, за конкретним співробітником, окремо для контактів

### Автоматизовані аналітики (телефонія)
- При інтеграції VoIP автоматично створюються:
  - **Звонок** (Дзвінок)
  - **Пропущений дзвінок**
- ⚠️ Кастомні поля додавати лише в КІНЦІ — щоб не зламати авто-заповнення

### Генерація документів з аналітик
- `Меню коментаря → Створити документ за шаблоном`
- Кілька коментарів можуть contributes до одного документа
- Приклади: квитанції, рахунки-фактури, пакувальні листи
