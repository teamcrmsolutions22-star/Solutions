# Доска контактов для собеседований | Как это сделать

в этом ролике покажем как план fix поможет упростить найм сотрудников в целом процесс простой заявки кандидатов поступают по почте через сайт или формы а также мессенджеры в итоге все заявки автоматически превращаются в удобные карточки задач с возможностью отслеживания статуса и обратной связью с кандидатом если в заявке указан имеil кандидата или она отправлена по почте система сама создаст карточку а когда кандидат будет одобрен бухгалтер получит к ней доступ оформит документы и переведёт его в сотрудники итак как же это всё настроить сначала оцифруем заявку кандидата в планфиксе с помощью специальной формы объекта на его основе система автоматически создаст задачу с информацией из заявки первым делом указываем название далее для того чтобы в задаче отображалась только необходимая информация настроим набор полей удаляем лишнение и добавляем нужные для удобства можно заранее расписать чек-лист для сотрудников для того чтобы у задач автоматически формировались названия используем переменные они автоматически подтянут в название значения из других полей объекта на этом же этапе указываем исполнителями сотрудников которые будут отвечать за общение с кандидатами переходим в набор статусов объекта чтобы настроить этапы отбора кандидатов удаляем неподходящие статусы оставляем только черновик и завершённое затем добавляем необходимые этапы указываем название и цвет таким же образом добавляем остальные этапы для удобства можно настроить переходы между ними когда этапы будут готовы по ним можно создать канбан теперь для коллективной работы с кандидатами настроим доступ к планировщику выбираем пункт который позволяет только выбранным сотрудникам использовать этот планировщик и указываем их и последнее настроим карточку кандидата где будут храниться его личные данные для этого создаём новый шаблон контакта для отображения только необходимой информации нужно настроить набор полей удалив лишнее для удобства поля можно перемещать например файлы оно пригодится для хранения резюме на этом этапе важно указать права доступа к контакту для HR-менеджера и бухгалтера чтобы они могли видеть и редактировать данные кандидатов первый этап настройки завершён осталось настроить получение заявок в план фикс как это сделать смотрите по ссылкам в описании автоматизируйте и упрощайте рабочие процессы вместе с Планфиксом
