# Поиск в Planfix | Как это сделать

поговорим о поиске в планфиксе Для этого в системе есть несколько инструментов Начнём с главного поисковой строки С её помощью можно искать объекты по части слова Планфикс покажет всё что начинается с сочетания указанных букв или по словоформам Тогда поиск покажет все объекты где использовалось указанное слово независимо от формы слова Кроме того искать можно сразу по нескольким словам Это удобно если вы помните только часть информации о задаче Для выбора нужного результата поиска можно использовать стрелки на клавиатуре А чтобы открыть нужную задачу нажмите Enter Ещё к поиску можно подключить значки избранного а также настроить группировку и сортировку результатов Кстати начинать процесс поиска можно без активации поисковой строки Так работает автофокусировка поиска Эту функцию можно активировать в настройках аккаунта И наконец чтобы быстро найти только что закрытую задачу можно использовать вкладку Недавние объекты Обязательно используйте все возможности поиска по системе чтобы ускорить и облегчить работу в планфиксе У
