# Управление проектами | Стандартные конфигурации Planfix

в этом видео покажем как работает конфигурация управления проектами в Планфиксе и как с её помощью автоматизировать и ускорить работу с проектами и задачами Конфигурация состоит из рабочего пространства в котором могут работать и сотрудники и руководитель проекта В первом разделе собраны все доступные пользователю проекты Для удобства они сразу разделены на активные и завершённые В бизнес-процессах большинство проектов типовые поэтому для ускорения их создания используются шаблоны Внутри шаблонов проектов есть шаблоны задач Это нужно чтобы при создании проекта автоматически создавались и задачи по нему Чтобы упростить создание первого шаблона предустановлен шаблон для поэтапной работы Редактируем его название Теперь меняем название шаблона Задаём название для задачи исполнителя и дедлайн И наконец добавляем конкретные требования в чек-лист Создадим ещё один шаблон задачи подобным способом Заполняем также название шаблона Указываем исполнителя название создаваемой задачи и дедлайн Включаем последовательность Теперь следующая задача начнётся только после завершения текущей Добавим к этой задаче подзадачу Заполняем необходимые поля и кликаем создать Таким же образом создаём ещё несколько подзадач Теперь посмотрим что получилось на примере этого шаблона проекта Названия дедлайны и исполнители задач подтянулись автоматически Добавим в таблицу столбец со статусами чтобы проверить сработала ли последовательность Всё отработало как надо Главная задача и первая подзадача в статусе новая а всё последующее в черновике Теперь при завершении текущей задачи для исполнителя будет автоматически ставиться следующее Пункты чек-листа также автоматически добавились к первой задаче Для удобства внутри проекта задачи можно отобразить в виде диаграммы Ганта При этом можно менять её масштаб и настройки отображения Чтобы видеть уведомления активные и завершённые задачи проекта можно включить вид планировщик Следующий раздел- общая диаграмма Тут настроена общая диаграмма по всем активным проектам Здесь же доступны фильтры по полям и настройки отображения Раздел моей задачи - основной рабочий планировщик сотрудников Каждый из них тут будет видеть свои уведомления календарь с запланированными задачами и почасовое планирование работы по ним на неделю Запланировать время работы можно обычным перетаскиванием задачи в календарь Также в планировщике можно увидеть все задачи распределённые по этапам выполнения Перейдём к планировщику руководителя Для сотрудников этот раздел недоступен Здесь руководитель может просматривать свои уведомления планировать задачи на неделю распределять своё рабочее время а также видеть общую загрузку сотрудников и важные задачи добавленные в избранное Рассмотрим раздел клиенты Он тоже доступен только для руководителя Для удобства все клиенты в разделе сразу поделены на физические и юридические лица Внутри раздела можно импортировать и экспортировать списки клиентов со всеми данными создавать новые контакты и компании переходить в карточки существующих настраивать столбцы с данными а также фильтровать информацию по ним И наконец последний раздел для руководителя Отчёты Здесь можно запускать готовые отчёты или создавать свои Помимо этого в раздел отдельно вынесены отчёты конфигурации например для отслеживания прогресса по проектам Также есть отчёты на основе данных которые сотрудники заполняют в задачах например план и факт по времени или бюджет Эти же данные можно вносить через аналитику настроенную в конфигурации Так данные по проекту всегда будут актуальны а отслеживать их можно всего в пару кликов Почти любой объект в Планфиксе можно добавить в избранное Для этого в конфигурации есть отдельный раздел Здесь удобно хранить проекты задачи отчёты и другие элементы к которым нужно быстро возвращаться На этом всё Используйте конфигурацию управления проектами чтобы упростить и ускорить запуск проектов управлять задачами и командой
