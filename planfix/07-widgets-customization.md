# Planfix — Віджети та програмні доробки

## Кастомні кнопки та дії

Адміністратор додає кнопки на панель картки задачі або контакту.

**Дії при натисканні:**
- Запуск автоматичного сценарію
- Відкриття зовнішнього iFrame-вікна
- Перехід за динамічним URL

**Контекст передачі даних:**
- GET-параметри: ID сутності, значення кастомних полів, токен авторизації

## Вбудовані сторінки та iFrame

- Сторонні веб-інтерфейси вбудовуються як кастомні вкладки або панелі планувальника
- **JavaScript SDK для iFrame → Planfix** не задокументований ⚠️
- Зв'язок реалізується через прямі REST API запити вбудованого додатка

## Обчислювані поля та формули

- **Математичні формули** для чисел та дат (тип поля «Розрахункове поле»)
- **Не підтримується:** виклик зовнішніх API з формули → для цього використовувати вебхуки в сценаріях

### Довідник формул

```
# Витягти домен з email
ПРОПИСН(REGEXPFIND({{email}}; "@([^.]+)\."))

# Витягти номер задачі з URL
REGEXPFIND(text; "(?<=/task/)\d+")

# Дата наступної п'ятниці
ДАТА(РІК(ЗАРАЗ());МІСЯЦЬ(ЗАРАЗ());ДЕНЬ(ЗАРАЗ())+ЯКЩО(ДЕНЬТИЖНЯ(ЗАРАЗ())<=5;5-ДЕНЬТИЖНЯ(ЗАРАЗ());5+7-ДЕНЬТИЖНЯ(ЗАРАЗ())))

# Переведення дати в інший часовий пояс (з червня 2025)
CONVERTTIMEZONE({{date}}; "Europe/Kiev")

# SHA3-256 хеш рядка (з січня 2026)
SHA3-256({{поле}})
```

### Доступні функції (повний список)

`ЯКЩО`, `ІНАКШЕ`, `ІНАКШЕЯКЩО`, `ЗЦІПИТИ`, `ОКРУГЛТ`, `REGEXPFIND`, `REGEXPREPLACE` (з січня 2026), `SHA3-256` (з січня 2026), `JSONARRAY`, `CONVERTTIMEZONE` (з червня 2025), `ДЛЯКОЖНОГО`, `ТІЛЬКОДАТА` та інші

### Перевірка синтаксису формул

- Доступна в користувацьких операціях сценаріїв (грудень 2025)

## Лайфхаки

### Перетягування дій у сценарії
Скопіювати блок дії → вставити в потрібному місці → видалити оригінал. (Прямого drag-and-drop немає.)

### QR-коди в документах (квітень 2026)
QR-коди доступні як файли з змінних задачі. Або: Python-скрипт FastAPI генерує QR → HTTP GET із ПФ → файл в інфоблок → поле задачі → змінна в шаблоні документа.
