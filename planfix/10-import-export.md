# Planfix — Імпорт / Експорт / Міграція

## Імпорт через UI

- **Об'єкти:** задачі, контакти, записи довідників
- **Формати:** CSV та Excel (XLS/XLSX)
- **Мапінг:** майстер імпорту зіставляє стовпці файлу з полями шаблонів
- **Перевірка дублів:** за Email або Телефон для контактів
- **Ліміти:** максимальна кількість рядків не задокументована ⚠️; обмежена розміром файлу

## Експорт даних

- «Керування акаунтом» → «Експорт даних»
- Повний експорт акаунта: бази об'єктів + прикріплені файли у вигляді архіву
- Доступний лише головному адміністратору

## Міграція з інших систем

| Система | Метод |
|---------|-------|
| **Trello** | Вбудований інструмент імпорту |
| **Asana та інші таск-менеджери** | Вбудовані інструменти імпорту |
| **Bitrix24** | Послідовний імпорт таблиць: спочатку контакти/компанії → потім проекти → задачі з прив'язкою |
| **amoCRM** | Аналогічно Bitrix24: контакти → проекти → задачі |

## Рекомендований порядок міграції з Bitrix24/amoCRM

1. Завантажити базу контактів та компаній
2. Завантажити проекти
3. Завантажити задачі з прив'язкою до імпортованих сутностей
