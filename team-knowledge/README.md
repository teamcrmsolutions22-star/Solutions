# 🏢 База знаний о компании — CRM Solutions

Внутренняя база знаний команды (регламенты, процессы, роли). Источник — Notion-пространство компании.
Деление по ролям и темам. Контент на рус/укр, имена папок — английские слаги (как во всём репозитории).

## Структура

| Папка | Раздел | Что внутри |
|---|---|---|
| [`general/`](general/) | **Общее** | О компании, виды проектов, инструменты, организация работы, команда и контакты |
| [`integrator/`](integrator/) | **Интегратор** (Исполнитель/настройщик) | Должностная инструкция, процесс работы, детальный процесс внедрения CRM, виды задач, гайд по анализу |
| [`account-pm/`](account-pm/) | **Аккаунт (ПМ)** | Зона ответственности клиент-менеджера/PM, контроль процессов |
| [`sales/`](sales/) | **Продажник** | Процессы продаж, прайс, калькуляторы КП, модель пакета часов |
| [`marketing/`](marketing/) | **Маркетинг** | Контент-процессы, SMM, обзоры виджетов, статьи |
| [`onboarding-hr/`](onboarding-hr/) | **Онбординг / HR** | Ввод в должность, портрет соискателя, правила работы, оплата, профайлинг |
| [`playbooks-templates/`](playbooks-templates/) | **Регламенты и шаблоны** | Правила игры с клиентом, документы клиента, скрипты обратной связи, алгоритм ТП |
| [`automations-planfix/`](automations-planfix/) | **Автоматизации Планфикс** | Карта триггеров/сценариев ПФ по стадиям |

## Служебное
- [`_pending-review.md`](_pending-review.md) — реестр материалов, которые пока не удалось изучить (видео/PDF/доки за allowlist'ом). Пополняется по ходу.

> Правило обновления: новый материал → определить раздел → дочитать существующий файл → добавить только новые факты (без дублей) → commit + push.
