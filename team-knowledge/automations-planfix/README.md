# Автоматизации Планфикс

Всё в аккаунте `crmsolutions.planfix.ru` (он же `.planfix.ua`). Корневая страница — «Автоматизации ПФ CRM Solutions».

## Активный проект (внедрение)
- **Ввод клиента** — триггер: «Предпродажная работа» → статус «Завершена» (с заполненным «Тип работы»; если не заполнен — уведомление Исполнителю/Аудитору).
- **Инициализация** — при создании проекта авто-кнопка «Инициализация»; клиенту в TG уходит план по неделям (1-я неделя — базовая настройка + дистанционное обучение по 2 ч/день; 2-я — обучение клиента 2–3 ч; доработки/ТП в 1 день первые 2 недели, далее 3 дня).
- **Настройка** — шаблон «Настройка Амосрм» (авто: 7 дней, Исполнитель, запрос данных, контрагент); поле «виджеты» → подзадачи; статус «В работе» → сообщение клиенту в TG.
- **Тестирование** — статус «Настройка → Выполнено» → уведомление клиенту; контроль через 2 / 4 / 6 дней.
- **Техподдержка** — статус «Защита → Завершено» → кнопка «ТП клиенту»: статус ТП + 14 дней, письмо на почту (правила игры ТП, видео, инструкция сотрудникам) + сообщение в TG; контроль через 3 и 10 дней.

## Анализ (предпродажа, серия «ПР: Анализ 1–9»)
Приветствие аналитика в чате → отправка гугл-формы → проверка полей (вид анализа/дата) с пингом менеджера в чат «Лиды» → проверка логин/пароль → напоминания клиенту за 2 ч / 1 ч / 15 мин / 1 мин (с регламентом и ссылкой на встречу) → после: «Выставить счёт» (проверка аналитики, вид работы = Анализ) → «Отправить клиенту» (отчёт + видео + факт-время на почту и в чат).

## Обучение клиента
Оповещения за 3 дня (пройти тесты), за 1 день / 15 мин / 5 мин (регламент, ссылка, демонстрация экрана), по окончании — итог работы.

## Техподдержка — команда `/ТП «описание»`
В чате Telegram клиента → создаётся подзадача в задаче ТП (дата+описание), уведомляется Исполнитель. Видно в планировщике ТП. Исполнитель трекает время + итог; «Завершить Исполнителем» → клиенту авто-уведомление о времени и итоге. **Контроль дедлайнов:** 1 / 2 / 3 (просрочено, TG нам) / 4 день (TG нам + клиенту).

## Выставление счёта за ТП
Проверить аналитику (описание, причина = тарификация, корректное время) → сгенерировать «Отчёт ТП» (столбец I = ценообразование часа) по проекту/дате → выгрузить XLSX по шаблону → в Google-папку «Оплата» → в задаче «Оплата» подзадача «Выставлен счёт» (дата от/до + ссылка + сумма; менеджеру в фильтр).

## Выставление чеков (еженедельно)
Каждую субботу клиенту в чат — время и сумма за 7 дней по Коммуникации/ТП/Настройке (старт — дата завершения задач «Коммуникация»/«ТП» на субботу; в пятницу PM проверяет аналитики). Только для проектов ПФ.
