# Инструменты в работе

Вся коммуникация с клиентом фиксируется (email / видео / аудио) и прикрепляется в систему учёта.

| Инструмент | Назначение |
|---|---|
| **Телеграм** | Мессенджер для общения внутри коллектива и рабочих групп с клиентом |
| **Планфикс** (https://crm--solutions.planfix.ua/) | Система учёта проектной работы: распределение обязанностей, файлы клиента, хронология действий, контроль исполнения |
| **Тайм-трекинг ПФ** (расширение Chrome «планфикс-таймер») | Учёт рабочего времени для Планфикс |
| **Гугл Диск** | Материалы от клиента и созданные по нему файлы (папки Настройка, Оплата) |
| **Компьютер** | Для работы |
| **AnyDesk** | Удалённое решение вопросов клиента |
| **Loom** | Запись экрана |
| **tl;dv** (tldv.io) | Запись видеовстреч в Google Meet |

Доступы клиента складываются в **Справочник клиента** в Планфикс (все доступы в одном месте), почта рассылок — `project@crmsolutions`.
