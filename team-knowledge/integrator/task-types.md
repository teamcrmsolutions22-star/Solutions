# Виды задач

## Общие виды задач
- **Повтори-ка (манки труд)** — повторить задачу по описанию/видеоинструкции в аккаунте клиента.
- **Коммуникация с Исполнителями** — решить задачу силами подрядчиков (коммуникация, пинг, контроль).
- **Тестирование системы** — воспроизвести бизнес-процесс клиента, скорректировать настройки, создать чек-лист.
- **ТП (техподдержка)** — организовать и контролировать решение задач клиента, быть на связи (телеграм/телефон).
- **Обучение сотрудников Заказчика** — на выезде или в видеообучении.
- **Проверка работы системы и сотрудников** — может входить в ТП; результат описывается в отчётах.
- **Настройка системы** — настроить систему по ТЗ/сценарию; отдельный человек тестирует на корректность.

## Виды задач отдела «Производство» (БД «Производство»)
Каждая задача = Название + Цель + ссылка на инструкцию.
- **Выполнять настройку проекта** — качественно, быстро, с пользой для клиента.
- **Искать виджеты / программные доработки / программистов** — не давать стандартное решение, а искать подходящее клиенту.
- **Описывать инструкцию для базы знаний** — контент для маркетинга + мануал для обучения/работы.
- **Описывать документирование анализа по проекту.**
- **Выполнять ТП клиентам** (пассивное и активное).
- **Вводить в должность (ментор по производству)** нового члена команды — сделать сильнее каждого.
- **Делать отчёт перед клиентом** и выяснять, как у них дела.

> Виды задач в разрезе отделов делятся на: Маркетинг (см. `../marketing/`), Производство (этот файл), Продажи (см. `../sales/`).
