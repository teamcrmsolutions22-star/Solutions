# Маркетинг

Источник: БД «Маркетинг» + «Процесс создания контента Производством». Руководитель маркетинга — Наташа.

## Задача отдела
**Создавать контент** (видео, инсайты по настройке, статьи) — контент для публикаций.
- Контент готовит **Производство**, 1 раз/неделю.
- Кейсы и статьи-мнения — раз в 2 недели (статья-мнение с участием Сергея).

## Категории контента
| Категория | Процесс |
|---|---|
| Видеообзор «Что умеет amoCRM» | запись обзора функций |
| Нюансы / Лайфхаки | Производство кидает лайфхаки в телеграм-группу (вопрос + скрин) |
| FAQ | вопрос + варианты ответа + правильный + пояснение, при необходимости видео |
| Обзор виджетов | фиксируется в Google-таблицу «Виджеты» + реферальная ссылка; тег маркетингу |
| Кейсы | 1 раз в 2 недели |
| Статья-мнение | темы → утверждение с руководителем → описание в Базе знаний; публикация на Tilda |
| Новинки amoCRM / CRM | мониторинг чатов партнёров (FB, Telegram), фиксация в документ/задачу |

## Каналы и инструменты
- Лайфхаки и виджеты фиксируются в Google-таблицах и телеграм-группе команды.
- Маркетинг дорабатывает (нарезка видео, музыка, текст) и публикует: Tilda, группы, личные страницы, База знаний **mybase.pro**.
