# Онбординг / HR

Ввод в должность, портрет соискателя, профайлинг, условия оплаты. Применимо при найме на любую роль (стартовая точка для новичка).

## Портрет соискателя / личные качества (Исполнитель)
- Готовность брать на себя ответственность, доводить задачи до результата.
- Любовь к порядку в документах; стремление к развитию и помощи клиентам.
- Вовлечённость, умение работать с обратной связью и давать её.
- Предлагать идеи по улучшению; переключаться между задачами; работать над ошибками.
- **Профайлинг:** шизоид, эпилептоид. Инициативный, обратная связь — факты, рефлексивный и проактивный.

## Ввод в должность (менторство по производству)
Отдельный вид задачи — «Вводить в должность (быть ментором по производству) нового члена команды». Цель — сделать сильнее каждого члена команды.
- Для ТП: перед допуском Исполнитель/Помощник изучает видеообучение по CRM и всем сервисам и **защищает знания перед Клиент-менеджером**; после сдачи — допуск к работе на ТП.

## Загрузка новичка
Руководитель даёт: ⅓ времени — обучение, ⅔ — выполнение работы. Недопустимо давать только обучение без оплаты (кроме стажировки).

## Оплата
- **Почасовая:** ставка за рабочий день (7 часов); среднее между плановым и фактическим (или фактическое).
- **Сдельная:** для задач со сложно считаемым временем; оплата предлагается вместе с задачей.

## Правила уважения в работе
Без эмоций/конфликтов/провокаций/оскорблений/высокомерия; не уходить, не закончив разговор; не пропадать со встреч/созвонов без предупреждения; не срывать сроки без предупреждения. Это касается всех ролей и обеих сторон (Руководитель ↔ Исполнитель).
