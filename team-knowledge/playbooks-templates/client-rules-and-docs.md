# Правила игры и документы клиента

## Правила игры
Документ, описывающий правила взаимоотношений между Заказчиком и Исполнителем. Высылается клиенту на email (ссылка + PDF, неизменяемые) до старта; есть видеообъяснение. Используется на Инициализации, ТП, Доработках.

## Документы клиента (высылаются до старта)
- **Правила игры** (взаимоотношения).
- **Документы по CRM/телефонии** — по каждой CRM свой документ (как работают системы и что ожидать).
- **Запрос метрик** (точка А) — гугл-форма, создаёт Интегратор; нужна для фиксации точки Б.
- **Документ «по желанию клиента»** — чего хочет Заказчик от работы.
- **Документ «До интервью»** — данные Заказчика до встречи.
- **Запрос данных для Амосрм/настройки** — генерируется из почты `project@crmsolutions`, отдаётся с видео и сроками.

**Важно:** документы называть по стандарту — для быстрого поиска в системе. Содержание оговаривается на встрече/по телефону (фиксировать аудио/видео).

## КП-шаблоны
Шаблон КП в Google Sheets, копируется под клиента. Подробно — `../sales/kp-calculators.md`.

## Хронология клиента
Документ, куда Клиент-менеджер вносит транскрибацию бизнес-процесса (даты, этапы, Д/З). Вкладки: База знаний, Хронология конкретного дня, Общая хронология, Д/З.
